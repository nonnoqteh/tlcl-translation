# فصل ۱۳ — سفارشی‌سازی پرامپت
### Chapter 13 – Customizing the Prompt

---

در این فصل به یک موضوع سرگرم‌کننده می‌پردازیم: **سفارشی‌سازی پرامپت** (prompt). پرامپت همان رشته‌ای است که هر بار قبل از اینکه دستوری وارد کنیم، نمایش داده می‌شود — مثل `[me@linuxbox ~]$`.

---

## پرامپت چگونه کار می‌کند؟
### Anatomy of a Prompt

پرامپت توسط متغیر محیطی `PS1` تعریف می‌شود (مخفف «Prompt String 1»). محتوایش را ببینیم:

```bash
echo $PS1
```

```
[\u@\h \W]\$
```

این رشته شامل **دنباله‌های فرار** (escape sequences) است که bash آن‌ها را جایگزین می‌کند:

| دنباله | جایگزین |
|--------|---------|
| `\a` | پخش صدای زنگ |
| `\d` | تاریخ به فرمت «Weekday Month Date» |
| `\h` | نام میزبان تا اولین نقطه |
| `\H` | نام میزبان کامل |
| `\j` | تعداد کارهای فعال |
| `\l` | نام دستگاه ترمینال |
| `\n` | خط جدید |
| `\r` | بازگشت کالسکه |
| `\s` | نام شل |
| `\t` | ساعت به فرمت ۲۴ ساعته (HH:MM:SS) |
| `\T` | ساعت به فرمت ۱۲ ساعته |
| `\@` | ساعت با AM/PM |
| `\u` | نام کاربری |
| `\v` | نسخه‌ی bash |
| `\V` | نسخه‌ی کامل bash |
| `\w` | مسیر پوشه‌ی کاری جاری |
| `\W` | نام پوشه‌ی کاری جاری (آخرین قسمت) |
| `\!` | شماره‌ی تاریخچه‌ی این دستور |
| `\#` | شماره‌ی دستور جاری |
| `\$` | `$` برای کاربر عادی، `#` برای root |
| `\\` | بک‌اسلش |
| `\[` | شروع رشته‌ی غیرچاپی |
| `\]` | پایان رشته‌ی غیرچاپی |

---

## ساختن پرامپت جدید

یک پرامپت ساده با اطلاعات بیشتر:

```bash
PS1="\u@\h:\w\$ "
echo $PS1
```

نتیجه:

```
me@linuxbox:~$ 
```

یا با نمایش ساعت:

```bash
PS1="\A \u@\h:\w\$ "
```

```
17:33 me@linuxbox:~$ 
```

## اضافه کردن رنگ
### Adding Color

ترمینال‌های مدرن از **کدهای فرار ANSI** برای رنگ پشتیبانی می‌کنند. ساختار:

```
\[\033[رنگ‌بخش‌ام\]
```

**رنگ‌های متن:**

| کد | رنگ |
|----|-----|
| `0;30` | سیاه |
| `0;31` | قرمز |
| `0;32` | سبز |
| `0;33` | قهوه‌ای/زرد |
| `0;34` | آبی |
| `0;35` | بنفش |
| `0;36` | آبی‌فیروزه‌ای |
| `0;37` | سفید |
| `1;30` | خاکستری تیره (bold سیاه) |
| `1;31` | قرمز روشن |
| `1;32` | سبز روشن |
| `1;33` | زرد |
| `1;34` | آبی روشن |
| `1;35` | بنفش روشن |
| `1;36` | فیروزه‌ای روشن |
| `1;37` | سفید |

**برای بازنشانی رنگ:** `\[\033[0m\]`

**مثال عملی — پرامپت سبز:**

```bash
PS1="\[\033[1;32m\]\u@\h:\w\$\[\033[0m\] "
```

**مثال با نام کاربری آبی و مسیر سبز:**

```bash
PS1="\[\033[1;34m\]\u@\h\[\033[0m\]:\[\033[1;32m\]\w\[\033[0m\]\$ "
```

**پرامپت برای root با رنگ قرمز** (هشدار بصری):

```bash
PS1="\[\033[1;31m\][\u@\h \W]\$\[\033[0m\] "
```

## اضافه کردن رنگ پس‌زمینه
### Background Colors

کدهای پس‌زمینه با `4` شروع می‌شوند:

| کد | رنگ پس‌زمینه |
|----|-------------|
| `40` | سیاه |
| `41` | قرمز |
| `42` | سبز |
| `43` | قهوه‌ای |
| `44` | آبی |
| `45` | بنفش |
| `46` | فیروزه‌ای |
| `47` | سفید |

## نگه‌داشتن پرامپت در تنظیمات

پس از پیدا کردن پرامپت ایده‌آل، آن را در `~/.bashrc` اضافه کن:

```bash
# پرامپت سفارشی
PS1="\[\033[1;32m\]\u@\h\[\033[0m\]:\[\033[1;34m\]\w\[\033[0m\]\$ "
export PS1
```

بعد از ذخیره:

```bash
source ~/.bashrc
```

---

## پرامپت‌های دیگر

علاوه بر `PS1`، متغیرهای پرامپت دیگری هم وجود دارند:

| متغیر | کاربرد |
|-------|--------|
| `PS1` | پرامپت اصلی |
| `PS2` | پرامپت ثانویه (وقتی دستور ادامه دارد) |
| `PS3` | پرامپت برای دستور `select` |
| `PS4` | پرامپت در حین اشکال‌زدایی |

مثال پرامپت ثانویه:

```bash
$ echo "چند
> خط"
```

`>` همان `PS2` است.

---

## خلاصه

سفارشی‌سازی پرامپت یکی از روش‌های جالب برای شخصی‌سازی تجربه‌ی لینوکس است. یک پرامپت خوب اطلاعات مفید نشان می‌دهد و در عین حال شلوغ نیست. تجربه کن و پرامپتی بساز که برایت راحت و کارآمد است.
