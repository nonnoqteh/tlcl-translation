# فصل ۲۲ — چاپ
### Chapter 22 – Printing

---

در این فصل با ابزارهای چاپ در خط فرمان آشنا می‌شویم:

- `pr` — تبدیل فایل‌های متنی برای چاپ
- `lpr` — چاپ فایل‌ها
- `lp` — چاپ فایل‌ها (System V)
- `a2ps` — قالب‌بندی فایل برای چاپ روی پرینتر PostScript
- `lpstat` — نمایش وضعیت چاپگر
- `lpq` — نمایش صف چاپ
- `lprm` / `cancel` — لغو کارهای چاپ

---

## تاریخچه‌ی کوتاه چاپ در یونیکس

در روزهای اولیه‌ی یونیکس، چاپ به شکل فیزیکی روی چاپگرهای خطی (line printers) انجام می‌شد. از آن زمان تا امروز، سیستم چاپ لینوکس تکامل زیادی داشته، اما ابزارهای خط فرمان آن تا حد زیادی ثابت مانده‌اند.

**سیستم چاپ CUPS** (Common Unix Printing System) در اکثر توزیع‌های مدرن لینوکس استفاده می‌شود.

---

## آماده‌سازی فایل برای چاپ

### `pr` — قالب‌بندی متن

`pr` متن ساده را برای چاپ قالب‌بندی می‌کند — صفحه‌بندی، هدر و فوتر اضافه می‌کند:

```bash
pr -l 60 -w 65 distros.txt
```

```
2025-06-09 11:00          distros.txt          Page 1


SUSE         10.2      12/07/2006
Fedora       10        11/25/2008
SUSE         11.0      06/19/2008
...
```

**گزینه‌های مفید `pr`:**

| گزینه | معنی |
|-------|------|
| `-l N` | تعداد خطوط صفحه (پیش‌فرض: ۶۶) |
| `-w N` | عرض صفحه |
| `-n` | شماره‌گذاری خطوط |
| `-N N` | چاپ چند ستونه |
| `+N` | شروع از صفحه‌ی N |
| `-h "text"` | هدر سفارشی |
| `-d` | دوبل‌فاصله |

---

## ارسال به چاپگر

### `lpr` — ارسال به صف چاپ (BSD)

```bash
lpr file.txt
```

با قالب‌بندی `pr`:

```bash
pr -l 60 distros.txt | lpr
```

**گزینه‌های `lpr`:**

| گزینه | معنی |
|-------|------|
| `-P printer` | انتخاب چاپگر |
| `-# N` | تعداد نسخه |
| `-p` | چاپ با عنوان استاندارد |

### `lp` — ارسال به صف چاپ (System V)

```bash
lp file.txt
```

```bash
lp -n 3 file.txt           # ۳ نسخه
lp -d printer_name file.txt # چاپگر مشخص
```

---

## مدیریت صف چاپ

### `lpstat` — وضعیت چاپگر

```bash
lpstat -a
```

```
printer1 accepting requests since Mon...
printer2 accepting requests since Tue...
```

```bash
lpstat -s    # خلاصه‌ی وضعیت
lpstat -p    # وضعیت چاپگرها
```

### `lpq` — صف چاپ

```bash
lpq
```

```
Rank    Owner   Job  File(s)          Total Size
active  me      603  test.txt         1024 bytes
1st     me      604  report.pdf       48293 bytes
```

### لغو کار چاپ

```bash
lprm 603         # BSD
cancel 603       # System V
cancel -a        # لغو همه‌ی کارها
```

## `a2ps` — تبدیل متن به PostScript

`a2ps` فایل متنی را به فرمت PostScript تبدیل می‌کند — با قابلیت‌های قالب‌بندی پیشرفته:

```bash
a2ps file.txt
```

```bash
a2ps -P printer file.txt          # ارسال مستقیم به چاپگر
a2ps --columns=2 file.txt         # دو ستونه
a2ps --output=file.ps file.txt    # ذخیره در فایل PS
```

**گزینه‌های مفید:**

| گزینه | معنی |
|-------|------|
| `--columns=N` | تعداد ستون |
| `--output=file` | ذخیره به فایل |
| `-P printer` | چاپگر |
| `--sides=1` | یک‌رو |
| `--sides=2` | دورو |
| `--landscape` | افقی |
| `--portrait` | عمودی |

---

## رابط گرافیکی CUPS

اگر CUPS نصب است، رابط وب آن در این آدرس دسترسی‌پذیر است:

```
http://localhost:631
```

از اینجا می‌توان چاپگرها را مدیریت کرد، صف چاپ را دید و تنظیمات پیشرفته انجام داد.

---

## خلاصه

ابزارهای چاپ لینوکس در خط فرمان کامل هستند. ترکیب `pr` با `lpr` یا `lp` برای اکثر نیازهای چاپ کافی است. `a2ps` برای کنترل دقیق‌تر قالب‌بندی بسیار مفید است.
