## توضیحات

متن توضیحی فصل اینجا قرار می‌گیرد. وقتی به یک دستور اشاره می‌کنید، آن را با backtick مشخص کنید: `ls`، `cd`، `pwd`.

---

## مثال‌ها

### دستور پایه

توضیح کوتاه اینکه این دستور چه کاری می‌کند:

```bash
ls -l /home
```

### چند دستور پشت هم

وقتی چند دستور به ترتیب اجرا می‌شوند:

```bash
cd /tmp
mkdir test-dir
ls -la
```

### خروجی نمونه

اگر بخواهید خروجی یک دستور را نشان دهید، از بلوک جداگانه استفاده کنید:

```bash
ls -l
```

```
total 32
drwxr-xr-x  2 user user 4096 Jun  9 10:00 Documents
drwxr-xr-x  2 user user 4096 Jun  9 10:00 Downloads
-rw-r--r--  1 user user  220 Jun  9 09:00 .bashrc
```

### اسکریپت

```bash
#!/bin/bash

# این یک اسکریپت نمونه است
NAME="دنیا"
echo "سلام، $NAME!"
```
