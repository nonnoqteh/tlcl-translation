# فصل ۲۱ — قالب‌بندی خروجی
### Chapter 21 – Formatting Output

---

در این فصل با ابزارهایی آشنا می‌شویم که برای کنترل نمایش متن طراحی شده‌اند:

- `nl` — شماره‌گذاری خطوط
- `fold` — شکستن خطوط به طول مشخص
- `fmt` — قالب‌بندی ساده‌ی پاراگراف
- `pr` — آماده‌سازی متن برای چاپ
- `printf` — قالب‌بندی و چاپ داده
- `groff` — سیستم قالب‌بندی اسناد

---

## `nl` — شماره‌گذاری خطوط

```bash
nl distros.txt | head
```

```
     1	SUSE        10.2
     2	Fedora      10
     3	SUSE        11.0
```

`nl` امکانات مختلف شماره‌گذاری دارد:

| گزینه | معنی |
|-------|------|
| `-b a` | شماره‌گذاری همه‌ی خطوط (شامل خالی) |
| `-b t` | شماره‌گذاری فقط خطوط غیرخالی (پیش‌فرض) |
| `-f a` | شماره‌گذاری footerها |
| `-h a` | شماره‌گذاری headerها |
| `-n rz` | اعداد با صفر جلو |
| `-w 3` | عرض شماره ۳ کاراکتر |
| `-i 2` | گام ۲ (۱، ۳، ۵، ...) |

---

## `fold` — شکستن خطوط بلند

```bash
echo "The quick brown fox jumped over the lazy dog." | fold -w 30
```

```
The quick brown fox jumped ove
r the lazy dog.
```

برای شکستن در مرز کلمه:

```bash
echo "The quick brown fox jumped over the lazy dog." | fold -w 30 -s
```

```
The quick brown fox jumped 
over the lazy dog.
```

---

## `fmt` — قالب‌بندی پاراگراف

`fmt` متن را با رعایت پاراگراف‌ها قالب‌بندی می‌کند:

```bash
fmt -w 50 file.txt
```

```bash
fmt -w 50 -p '# ' file.txt    # حفظ پیشوند کامنت
```

---

## `pr` — آماده‌سازی برای چاپ

`pr` صفحه‌بندی می‌کند و هدر/فوتر اضافه می‌کند:

```bash
pr -l 20 -w 70 file.txt | head -40
```

```
2025-01-01 10:00        distros.txt        Page 1


SUSE        10.2
Fedora      10
...
```

**گزینه‌های مفید `pr`:**

| گزینه | معنی |
|-------|------|
| `-l N` | طول صفحه N خط |
| `-w N` | عرض صفحه N کاراکتر |
| `-n` | شماره‌گذاری خطوط |
| `-N N` | تعداد ستون‌ها |
| `+N` | شروع از صفحه N |

---

## `printf` — قالب‌بندی پیشرفته

`printf` (مثل C) خروجی قالب‌بندی‌شده چاپ می‌کند:

```bash
printf "I formatted the string: %s\n" foo
```

```
I formatted the string: foo
```

**مشخص‌کننده‌های تبدیل:**

| مشخص‌کننده | توضیح |
|-----------|-------|
| `%d` | عدد صحیح |
| `%f` | عدد اعشاری |
| `%e` | علمی |
| `%s` | رشته |
| `%c` | کاراکتر |
| `%%` | کاراکتر `%` |

مثال‌ها:

```bash
printf "%d, %f, %s\n" 380 380 380
```

```
380, 380.000000, 380
```

```bash
printf "%-20s%05d\n" "me" 42
```

```
me                  00042
```

توضیح فرمت:
- `-20s` = رشته‌ی چپ‌چین با عرض ۲۰
- `05d` = عدد صحیح با صفر جلو و عرض ۵

---

## خلاصه

قالب‌بندی خروجی شاید کمتر استفاده شود، اما وقتی لازم است — مثلاً آماده‌سازی گزارش، ساختن خروجی تمیز از اسکریپت، یا پردازش داده‌ی جدولی — این ابزارها فوق‌العاده کارآمدند. `printf` به‌خصوص در اسکریپت‌نویسی بسیار پرکاربرد است.
