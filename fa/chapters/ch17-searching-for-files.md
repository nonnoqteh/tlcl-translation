# فصل ۱۷ — جستجو در فایل‌ها
### Chapter 17 – Searching for Files

---

دستورات این فصل:

- `locate` — جستجوی فایل با نام
- `find` — جستجوی فایل در درخت پوشه
- `xargs` — ساختن و اجرای دستور از ورودی استاندارد
- `touch` — تغییر زمان فایل
- `stat` — نمایش اطلاعات فایل و سیستم فایل

---

## `locate` — جستجوی سریع

`locate` در یک **پایگاه داده** از نام فایل‌ها جستجو می‌کند که از قبل ساخته شده:

```bash
locate bin/zip
```

```
/usr/bin/zip
/usr/bin/zipcloak
/usr/bin/zipgrep
/usr/bin/zipinfo
/usr/bin/zipnote
/usr/bin/zipsplit
```

می‌توانیم با لوله‌کشی به `grep` نتایج را فیلتر کنیم:

```bash
locate zip | grep bin
```

> **نکته:** پایگاه داده‌ی `locate` معمولاً یک‌بار در شبانه‌روز به‌روز می‌شود. فایل‌های تازه‌ساخته‌شده ممکن است پیدا نشوند. برای به‌روزرسانی دستی:
> ```bash
> sudo updatedb
> ```

## `find` — جستجوی پیشرفته

`find` در زمان واقعی در سیستم فایل جستجو می‌کند — کندتر از `locate` اما بسیار قدرتمندتر.

### جستجوی ساده

```bash
find ~
```

فهرست همه‌ی فایل‌های پوشه‌ی خانگی را می‌دهد.

```bash
find ~ | wc -l
```

```
47068
```

### جستجو بر اساس نوع

```bash
find ~ -type f    # فقط فایل‌های عادی
find ~ -type d    # فقط پوشه‌ها
find ~ -type l    # فقط لینک‌های نمادین
```

### جستجو بر اساس نام

```bash
find ~ -name "*.jpg"
find ~ -iname "*.jpg"    # بدون توجه به بزرگی/کوچکی
```

### جستجو بر اساس اندازه

```bash
find ~ -size +1M          # بزرگ‌تر از ۱ مگابایت
find ~ -size -10k         # کوچک‌تر از ۱۰ کیلوبایت
find ~ -size +100M        # بزرگ‌تر از ۱۰۰ مگابایت
```

واحدها: `b` (بلاک ۵۱۲ بایتی)، `c` (بایت)، `k` (کیلوبایت)، `M` (مگابایت)، `G` (گیگابایت)

### جستجو بر اساس زمان

```bash
find ~ -mtime -3        # تغییریافته در ۳ روز گذشته
find ~ -newer file.txt  # جدیدتر از file.txt
```

### جستجو بر اساس مجوز

```bash
find ~ -perm 0600        # دقیقاً مجوز 0600
find ~ -perm /0222       # هر بیت نوشتن تنظیم‌شده
find ~ -perm -0600       # حداقل مجوز 0600
```

### جستجو بر اساس کاربر/گروه

```bash
find ~ -user me           # متعلق به کاربر me
find ~ -group staff       # متعلق به گروه staff
find /tmp -nouser         # فایل‌های بدون کاربر
```

### عملگرهای منطقی

```bash
find ~ -type f -name "*.jpg"   # AND (پیش‌فرض)
find ~ -name "*.jpg" -o -name "*.png"  # OR
find ~ -not -name "*.jpg"      # NOT
find ~ ! -name "*.jpg"         # NOT (شکل مختصر)
```

## اجرای عملیات روی نتایج
### Running Commands

### `find -exec` — اجرای مستقیم

```bash
find ~ -type f -name "*.jpg" -exec ls -lh {} \;
```

- `{}` جایگزین هر نام فایل پیداشده می‌شود
- `\;` پایان دستور را مشخص می‌کند

برای عملکرد سریع‌تر (ارسال همه به یکجا):

```bash
find ~ -type f -name "*.jpg" -exec ls -lh {} +
```

### پیدا و حذف کردن فایل‌های مشخص

```bash
# پیدا کردن فایل‌های tmp (اول نگاه کنید چه پیدا می‌شود)
find /tmp -name "tmp*" -type f

# بعد حذف
find /tmp -name "tmp*" -type f -delete
```

> ⚠️ با `-delete` مراقب باشید! اول بدون `-delete` اجرا کنید تا ببینید چه فایل‌هایی پیدا می‌شود.

---

## `xargs` — ساختن دستور از ورودی

`xargs` لیست آیتم‌ها را از stdin می‌خواند و آن‌ها را به عنوان آرگومان دستور دیگری ارسال می‌کند:

```bash
find ~ -type f -name "*.jpg" | xargs ls -lh
```

برای فایل‌هایی که نامشان فاصله دارد:

```bash
find ~ -type f -name "*.jpg" -print0 | xargs --null ls -lh
```

---

## `touch` — تغییر زمان فایل

```bash
touch file.txt
```

اگر فایل وجود نداشته باشد، خالی ایجاد می‌شود. اگر وجود داشته باشد، زمان آن به‌روز می‌شود.

## خلاصه

`find` یکی از قدرتمندترین ابزارهای لینوکس است. ترکیب آن با `-exec` یا `xargs` امکانات بی‌شماری می‌دهد. وقتی نیاز داری فایل‌هایی با ویژگی‌های خاص پیدا کنی و روی آن‌ها عملیات انجام دهی، `find` ابزار توست.
