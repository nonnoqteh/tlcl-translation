# ترجمه‌ی فارسی — خط فرمان لینوکس

این پوشه حاوی ترجمه‌ی فارسی فصل‌های کتاب است.

## راهنمای ترجمه‌گر

- هر فصل یک فایل جداگانه در `chapters/` دارد
- نام فایل‌ها: `ch01-what-is-the-shell.md`، `ch02-navigation.md` و ...
- در ابتدای هر فایل، عنوان فصل به فارسی و انگلیسی نوشته شود
- کدهای شل و دستورها **ترجمه نمی‌شوند**، فقط توضیحات‌شان
- واژه‌های فنی: ابتدا معادل فارسی، سپس انگلیسی در پرانتز — مثلاً: `پوشه (directory)`

## واژه‌نامه‌ی پرکاربرد

| انگلیسی | فارسی |
|---------|-------|
| shell | شل |
| command | دستور |
| directory | پوشه |
| file | فایل |
| process | فرآیند |
| permission | مجوز |
| terminal | پایانه |
| prompt | فراخوان |
| argument | آرگومان |
| option / flag | گزینه |
| standard input | ورودی استاندارد |
| standard output | خروجی استاندارد |
| pipeline | لوله‌کشی |
| redirection | تغییر مسیر |
| environment variable | متغیر محیطی |
