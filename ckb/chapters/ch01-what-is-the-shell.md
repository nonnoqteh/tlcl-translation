# بابەتی ١ — شێل چییە؟
### Chapter 1 – What Is the Shell?

---

کاتێک دەربارەی «ڕووکاری فەرمانی» (command line interface) قسە دەکەین، مەبەستمان ئەوەیە کە شێل بوەست. **شێل** (shell) بەرنامەیەکی تایبەتییە کە ئینپوتی کیبۆردت وەردەگرێت و بە سیستەمی کارپێکردن ڕادەگەیەنێت. بەپێچەوانەی **ڕووکاری گرافیکی** (GUI)، شێل ڕووکاری **دەقی** (text-based) یە.

بەشەکانی ئەم بابەتە:
- ئیمیوولاتەری تێرمینال
- یەکەم پەنجەدانەکان
- مێژووی فەرمان
- فەرمانە سادەکان

---

## ئیمیوولاتەری تێرمینال
### Terminal Emulators

بۆ بەکارهێنانی شێل، پێویستمان بە **ئیمیوولاتەری تێرمینال** (terminal emulator) هەیە. زۆرێک لە توزیعەکانی لینوکس بەرنامەیەکی تێرمینالیان هەیە:
- **GNOME Terminal** — لە ژینگەی GNOME
- **Konsole** — لە ژینگەی KDE
- **xterm** — ئیمیوولاتەری کلاسیکی

---

## یەکەم پەنجەدانەکان
### First Keystrokes

کاتێک تێرمینال دەکرێتەوە، دەبینین:

```
[me@linuxbox ~]$
```

ئەمە **نیشانی شێل** (shell prompt) ە. مەبەستی:
- `me` — ناوی بەکارهێنەر
- `linuxbox` — ناوی کۆمپیوتەر
- `~` — پووشپەری ماڵ (home directory)
- `$` — بەکارهێنەری ئاسایی (ئەگەر `#` بوو، سوپەریوزەرە)

---

## تاریخچەی فەرمان
### Command History

bash دواین **١٠٠٠** فەرمانی بەکارهاتوو لە فایلی `~/.bash_history` دەپارێزێت.

بۆ گەڕانەوەی فەرمانە کۆنەکان:
- **تیر بەرەو سەرەوە (↑)** — فەرمانی پێشوو
- **تیر بەرەو خوارەوە (↓)** — فەرمانی داهاتوو

---

## جووڵانی کەرسەر
### Cursor Movement

| کلیل | کارکردن |
|------|--------|
| `Ctrl-a` | کۆچ بۆ سەرەتای هێڵ |
| `Ctrl-e` | کۆچ بۆ کۆتایی هێڵ |
| `Ctrl-b` | یەک پیت دواوە |
| `Ctrl-f` | یەک پیت پێشوو |
| `Alt-b` | یەک وشە دواوە |
| `Alt-f` | یەک وشە پێشوو |

## کۆتایی هێنانی نیشستی تێرمینال
### Ending a Terminal Session

بۆ داخستنی نیشست:

```bash
exit
```

یان **Ctrl-d** بفشار.

---

## کونسۆلی مەجازی
### Virtual Consoles

زۆری لە توزیعەکانی لینوکس **٦ کونسۆلی مەجازی** (virtual consoles) هەیانە کە دەتوانرێت بە **Ctrl-Alt-F1** تا **Ctrl-Alt-F6** پێیان بگەیت. بۆ گەڕانەوە بۆ ئیمیوولاتەری گرافیکی: **Ctrl-Alt-F7**.
