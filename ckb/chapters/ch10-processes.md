# بابەتی ١٠ — پرۆسەکان
### Chapter 10 – Processes

---

سیستەمی کارپێکردنی مۆدێرن بە بنچینە **زۆرکارییانە** (multitasking) ن — خەیاڵێکی جێبەجێکردنی هاوکاتی چەند بەرنامە بە گۆڕانی خێرای لەنێوانیاندا دروستدەکەن. هسته‌ی لینوکس ئەم کارە لە ڕێگەی **پرۆسەکان** (processes) بەڕێدەبات.

فەرمانەکانی ئەم بابەتە:
- `ps` — نیشاندانی پرۆسەی ئێستا
- `top` — نیشاندانی دینامیکی پرۆسەکان
- `jobs` — فێرستکردنی کارەکانی چالاک
- `bg` — بەردەوامکردنی کار لە پشتزمین
- `fg` — بەردەوامکردنی کار لە پێشزمین
- `kill` — ناردنی سیگنال بۆ پرۆسە
- `killall` — ناردنی سیگنال بۆ چەند پرۆسە
- `shutdown` — خاموشکردن یان دەستپێکردنەوەی سیستەم

---

## پرۆسە چۆن کار دەکات؟

کاتێک سیستەم دەستی دەکات، هسته چەند **پرۆسەی دیمۆن** (daemon processes) دروستدەکات کە لە پشتزمین کار دەکەن. بەرنامەی بەکارهێنەریش پرۆسەی خۆیان دروستدەکەن.

هسته بۆ هەر پرۆسە **ناسنامەی پرۆسە** (PID — Process ID) تایبەت دەکات. PIДی یەکەم پرۆسەی سیستەم `init` ە بە PID 1.

## بینینی پرۆسەکان

### `ps` — وێنەی پرۆسەکان

```bash
ps
```

```
  PID TTY          TIME CMD
 5198 pts/1    00:00:00 bash
10129 pts/1    00:00:00 ps
```

بۆ بینینی زیاتر:

```bash
ps x
```

```
  PID TTY      STAT   TIME COMMAND
 2799 ?        Ssl    0:00 /usr/libexec/bonobo-activation-server...
10133 pts/1    R+     0:00 ps x
```

**ستوونی STAT (دۆخی پرۆسە):**

| کۆد | مانا |
|-----|------|
| `R` | لە جێبەجێکردندایە (Running) |
| `S` | نوێستوو (Sleeping) — چاوەڕوانی ڕووداو |
| `D` | نوێستووی ناڕووخاو — چاوەڕوانی I/O |
| `T` | وەستاو (Stopped) |
| `Z` | زۆمبی — پرۆسەی مردوو کە هێشتا پاکنەبووەوە |
| `<` | ئەولەویەتی بەرز |
| `N` | ئەولەویەتی نزم |

بۆ نیشاندانی هەموو پرۆسەی سیستەم:

```bash
ps aux
```

### `top` — نیشاندانی دینامیکی پرۆسەکان

`top` زانیارییەکان بە شێوازی **زیندوو** نوێدەکاتەوە (بە بنچینە: هەر ٣ چرکە):

```bash
top
```

```
top - 14:59:20 up 6:30,  2 users,  load average: 0.07, 0.02, 0.00
Tasks: 109 total,   1 running, 106 sleeping,   0 stopped,   2 zombie
Cpu(s):  0.7%us,  1.0%sy,  0.0%ni, 98.3%id
Mem:    319496k total,   314860k used,     4636k free
```

**کلیلەکانی `top`:**

| کلیل | کارکردن |
|------|--------|
| `h` | یارمەتی |
| `q` | دەرچوون |
| `M` | ڕیزکردن بەپێی بیرگە |
| `P` | ڕیزکردن بەپێی CPU |
| `k` | کوشتنی پرۆسە (بە داخلکردنی PID) |

## کۆنترۆڵکردنی پرۆسەکان

### جێبەجێکردن لە پشتزمین (Background)

بە زیادکردنی `&` بۆ کۆتایی فەرمان:

```bash
xlogo &
```

```
[1] 28236
```

### `jobs` — فێرستی کارەکانی چالاک

```bash
jobs
```

```
[1]+ Running                 xlogo &
```

### `fg` — گەڕانەوە بۆ پێشزمین

```bash
fg %1
```

### وەستاندنی پرۆسە (Ctrl-z)

(Ctrl-z بفشار)

```
[1]+ Stopped                 xlogo
```

پرۆسە **وەستا** بەڵام **نەکوژرا**. بۆ بەردەوامکردن لە پشتزمین:

```bash
bg %1
```

## سیگنالەکان و `kill`

`kill` سیگنال دەنێرێت، نەک هەمیشە «دەکوژێت»:

```bash
kill [-signal] PID...
```

**سیگنالە گرنگەکان:**

| ژمارە | ناو | مانا |
|-------|-----|------|
| 1 | `HUP` | بڕانی پەیوەندی — بۆ دەستپێکردنەوەی پرۆسەی دیمۆن |
| 2 | `INT` | ڕووخان — وەک Ctrl-c |
| 9 | `KILL` | کوشتنی خێرا لەلایەن هسته (ناڕووخێنراو) |
| 15 | `TERM` | کۆتایی — بنچینەیی `kill` |
| 18 | `CONT` | بەردەوامکردن — دوای STOP |
| 19 | `STOP` | وەستان (ناڕووخێنراو) |
| 20 | `TSTP` | وەستانی تێرمینال — وەک Ctrl-z |

```bash
kill -9 28236           # KILL — کوشتنی زەبری
kill -SIGKILL 28236     # بە ناوی تەواو
kill 28236              # TERM — بنچینەیی
```

---

### `killall` — ناردنی سیگنال بۆ چەند پرۆسە

بۆ کوشتنی هەموو پرۆسەی بە یەک ناو:

```bash
killall xlogo
```

## بەڕێوەبردنی ئەولەویەت

لینوکس بۆ هەر پرۆسە **ئەولەویەتی کاتبەندی** دەدات. ماوەکە لە **-20** (بەرزترین ئەولەویەت) تا **19** (نزمترین) ە.

بۆ دەستپێکردنی پرۆسە بە ئەولەویەتی نزمتر:

```bash
nice -n 10 cpu-intensive-command
```

بۆ گۆڕینی ئەولەویەتی پرۆسەی لە جێبەجێکردندا:

```bash
renice 15 -p 28236
```

> **تێبینی:** بەکارهێنەری ئاسایی تەنها دەتوانێت ئەولەویەت **نزمتر** بکات. بەرزکردنەوەی ئەولەویەت پێویستیی سوپەریوزەری هەیە.

## خاموشکردنی سیستەم

```bash
sudo reboot                 # دەستپێکردنەوە
sudo shutdown -h now        # خاموشکردنی خێرا
sudo shutdown -r now        # دەستپێکردنەوەی خێرا
sudo shutdown -h +10        # خاموشکردن دوای ١٠ خولەک
```

لەم بابەتەدا ئامرازە بنچینەییەکانی بینین و کۆنترۆڵکردنی پرۆسەکان بینیمان. بە `ps` و `top` دۆخی سیستەم بپشکنین، بە `kill` و `killall` پرۆسەکان کۆنترۆڵ بکەن، و بە `&`، `fg`، `bg` و `jobs` کارەکان لەنێوان پێشزمین و پشتزمین جووڵاندن.
