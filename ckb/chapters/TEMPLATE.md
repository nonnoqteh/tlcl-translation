## ڕوونکردنەوە

دەقی ڕوونکردنەوەی باپەڕەکە لێرەدا دادەنرێت. کاتێک بە فەرمانێک ئاماژە دەدەیت، بە backtick دیاریش بکە: `ls`، `cd`، `pwd`.

---

## نموونەکان

### فەرمانی بنچینەیی

ڕوونکردنەوەی کورت کە ئەم فەرمانە چی دەکات:

```bash
ls -l /home
```

### چەند فەرمان پشت یەکەوە

کاتێک چەند فەرمان بە ریزەوە جێبەجێ دەکرێن:

```bash
cd /tmp
mkdir test-dir
ls -la
```

### دەرچوونی نموونەیی

ئەگەر بتەوێت دەرچوونی فەرمانێک پیشان بدەیت، بلۆکێکی جیاواز بەکاربهێنە:

```bash
ls -l
```

```
total 32
drwxr-xr-x  2 user user 4096 Jun  9 10:00 Documents
drwxr-xr-x  2 user user 4096 Jun  9 10:00 Downloads
-rw-r--r--  1 user user  220 Jun  9 09:00 .bashrc
```

### سکریپت

```bash
#!/bin/bash

# ئەمە سکریپتێکی نموونەیییە
NAME="جیهان"
echo "سڵاو، $NAME!"
```
