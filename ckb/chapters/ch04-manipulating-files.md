# بابەتی ٤ — کارکردن لەگەڵ فایل و فۆڵدەر
### Chapter 4 – Manipulating Files and Directories

---

فەرمانەکانی ئەم بابەتە:

- `cp` — کۆپیکردنی فایل و فۆڵدەر
- `mv` — جووڵاندن / ناودەستکردنەوەی فایل و فۆڵدەر
- `mkdir` — دروستکردنی فۆڵدەر
- `rm` — سڕینەوەی فایل و فۆڵدەر
- `ln` — دروستکردنی لینکی زەق و سیمبۆلی

---

## جۆکەرەکان (Wildcards)

پێش سەردانی فەرمانەکان، پێویستە جۆکەرەکان بناسین — چونکە زۆر گرنگن.

| جۆکەر | مانا |
|-------|------|
| `*` | هەر زنجیرەی پیتێک (تووخم بوون یان بوون) |
| `?` | هەر پیتێکی تەک |
| `[پیتەکان]` | هەر پیتێک لە کۆمەڵێکدا |
| `[!پیتەکان]` | هەر پیتێک **نە** لە کۆمەڵدا |
| `[[:class:]]` | هەر پیتێک لە پۆلی دیاریکراودا |

**پۆلەکانی بەکارهراو:**

| پۆل | مانا |
|-----|------|
| `[:alnum:]` | هەر پیت یان ژمارەیەک |
| `[:alpha:]` | هەر پیتێک |
| `[:digit:]` | هەر ژمارەیەک |
| `[:lower:]` | هەر پیتی بچووکێک |
| `[:upper:]` | هەر پیتی گەورەیەک |

**نموونەکان:**

| ئالگۆ | لێدانی |
|-------|--------|
| `*` | هەموو فایلەکان |
| `g*` | فایلی دەستپێکراو بە `g` |
| `b*.txt` | دەستپێکراو بە `b`، کۆتایی بە `.txt` |
| `Data???` | `Data` بەدوای سێ پیتی هەر |
| `[abc]*` | دەستپێکراو بە `a`، `b` یان `c` |
| `[[:upper:]]*` | دەستپێکراو بە پیتی گەورە |
| `*[[:digit:]]` | کۆتایی بە ژمارە |

## `mkdir` — دروستکردنی فۆڵدەر

```bash
mkdir dir1
mkdir dir1 dir2 dir3
mkdir -p dir1/dir2/dir3    # دروستکردنی ئەوەی پێویستە
```

---

## `cp` — کۆپیکردن

```bash
cp item1 item2
cp item... directory
```

**ئۆپشنەکانی گرنگ:**

| ئۆپشن | مانا |
|-------|------|
| `-a` | کۆپیکردن بە هەموو تایبەتمەندییەکانیەوە |
| `-i` | پرسیار پێش دەستکاریکردن |
| `-r` | کۆپیکردنی بەرفراوانی فۆڵدەر |
| `-u` | تەنها کۆپیکردنی فایلی نوێ یان گۆڕدراو |
| `-v` | نیشاندانی پەیامی تفصیلی |

**نموونەکان:**

```bash
cp file1 file2                    # کۆپیکردنی file1 بۆ file2
cp -i file1 file2                 # پرسیار پێش دەستکاری
cp file1 file2 dir1               # کۆپی بە dir1
cp dir1/* dir2                    # کۆپیکردنی هەموو ناوەرۆکی dir1 بۆ dir2
cp -r dir1 dir2                   # کۆپیکردنی بەرفراوانی dir1
```

---

## `mv` — جووڵاندن / ناودەستکردنەوە

```bash
mv item1 item2
mv item... directory
```

**ئۆپشنەکانی گرنگ:**

| ئۆپشن | مانا |
|-------|------|
| `-i` | پرسیار پێش دەستکاریکردن |
| `-u` | تەنها جووڵاندنی فایلی نوێ یان گۆڕدراو |
| `-v` | نیشاندانی پەیامی تفصیلی |

**نموونەکان:**

```bash
mv file1 file2              # ناودەستکردنەوە
mv file1 file2 dir1         # جووڵاندن بۆ dir1
mv dir1 dir2                # ئەگەر dir2 نەبوو: ناو دەستکاردەکات
```

---

## `rm` — سڕینەوە

```bash
rm item...
```

**ئۆپشنەکانی گرنگ:**

| ئۆپشن | مانا |
|-------|------|
| `-i` | پرسیار پێش سڕینەوە |
| `-r` | سڕینەوەی بەرفراوانی فۆڵدەر |
| `-f` | سڕینەوەی بەبێ پرسیار |
| `-v` | نیشاندانی تفصیل |

> ⚠️ **ئاگاداری گرنگ:** لینوکس هیچ **سەبەتەی ئەخساندنی** (Recycle Bin) نیە! فایلی سڕاوە **بەهیچ شێوەیەک** بەدەستناهێنرێتەوە. پێش بەکارهێنانی `rm`، بە `ls` سەرت بدەرێت.

```bash
rm -rf dir1 dir2            # سڕینەوەی بەرفراوانی بەبێ پرسیار
```

---

## `ln` — دروستکردنی لینک

### لینکی زەق (Hard Link)

```bash
ln file1 link1
```

لینکی زەق دووەم ناوێکە بۆ هەمان فایل. ئەگەر ئەسلەکە بسڕدرێتەوە، لینک هێشتا کار دەکات.

### لینکی سیمبۆلی (Symbolic Link)

```bash
ln -s file1 link1
ln -s /usr/bin link_to_usr_bin
```

لینکی سیمبۆلی مانای ژامووی ویندۆز (shortcut) دەدات. `l` لە `ls -l` دا دەیبینین.

## ڕاهێنانی کاری: گرووپی بازیچە

پڕاکتیس بکە بە دروستکردنی ژینگەی تاقیکردنەوە:

```bash
mkdir -p playground/dir1 playground/dir2
cp /etc/passwd playground/dir1
cd playground
cp dir1/passwd dir2
cp -i dir1/passwd dir2       # پرسیار لەسەر دەستکاری
ls -l dir1 dir2
mv dir1/passwd .
ls dir1 dir2
mv passwd dir1
mv dir1/passwd dir2/file1
mv dir2/file1 dir2/file2
mkdir dir3
mv dir2/file2 dir3
ls dir3
cd dir3
rm file2                     # ئەگەر ئەمە بروو: rm dir3/file2
cd ..
rm -r dir3
```
