# بابەتی ٣ — گەڕانەوەی سیستەم
### Chapter 3 – Exploring the System

---

لەم بابەتەدا بەرفراوانتر سیستەم دەگەڕێینەوە. فەرمانە نوێیەکان:

- `ls` — فێرست کردن (بە زیاتر ئۆپشن)
- `file` — دیاریکردنی جۆری فایل
- `less` — خوێندنەوەی فایلی دەق

---

## `ls` — تواناکانی زیاتر

### ئۆپشنی `-l` — فۆرماتی درێژ

```bash
ls -l
```

```
total 56
drwxr-xr-x  2 me me  4096 2025-01-01 Desktop
drwxr-xr-x  2 me me  4096 2025-01-01 Documents
-rw-r--r--  1 me me 58234 2025-02-14 ls-output.txt
```

**مانای هەر ستوونێک:**

| ستوون | مانا |
|-------|------|
| `drwxr-xr-x` | مۆڵەتەکان |
| `2` | ژمارەی لینکی زەق |
| `me` | خاوەن |
| `me` | گروپ |
| `4096` | قەبارە بە بایت |
| `2025-01-01` | کاتی گۆڕانکاری دوایین |
| `Desktop` | ناو |

### بەکارهێنانی چەندین ئۆپشن

```bash
ls -lt         # مرتبکردن بەپێی کات
ls -lt --reverse  # مرتبکردنی بەپێچەوانە
ls -a          # نیشاندانی فایلە شاراوەکان
ls -lh         # قەبارە بە فۆرماتی خوێندراو
```

---

## `file` — دیاریکردنی جۆری فایل

```bash
file picture.jpg
```

```
picture.jpg: JPEG image data, JFIF standard 1.01
```

```bash
file /etc/passwd
```

```
/etc/passwd: ASCII text
```

> لینوکس بەپێی **ناوەرۆکی فایل** جۆرەکەی دیاری دەکات، نەک پسوندەکە.

---

## `less` — خوێندنەوەی فایل

```bash
less /etc/passwd
```

**کلیلەکانی `less`:**

| کلیل | کارکردن |
|------|--------|
| `PgUp` یان `b` | یەک لاپەڕە بەرەو سەرەوە |
| `PgDn` یان `Space` | یەک لاپەڕە بەرەو خوارەوە |
| `G` | کۆچ بۆ کۆتایی فایل |
| `g` | کۆچ بۆ سەرەتای فایل |
| `/وشە` | گەڕان بۆ پێشوو |
| `n` | گەڕان بۆ دووبارەکردنەوەی داهاتوو |
| `q` | دەرچوون |

---

## گەشتی ئامۆژگارانە — پووشپەرە گرنگەکانی لینوکس

| پووشپەر | ناوەرۆک |
|---------|---------|
| `/` | پووشپەری کۆک — سەرەتای هەموو شتێک |
| `/bin` | فەرمانە بنچینەیییەکان (`cp`, `ls`, `mv`) |
| `/boot` | هسته‌ی لینوکس و فایلە پەیوەستەکانی بووتکردن |
| `/dev` | نۆدی ئامێر |
| `/etc` | فایلە ڕێکخستنی سیستەم |
| `/home` | پووشپەری ماڵی بەکارهێنەران |
| `/lib` | کتێبخانەی هاوبەشی هسته |
| `/media` | خاڵی سواربوون بۆ میدیای گۆڕاو |
| `/mnt` | خاڵی سواربوونی کەیفی |
| `/opt` | نرمەواڵەی ئۆپشنی |
| `/proc` | سیستەمی فایلی مەجازی هسته |
| `/root` | پووشپەری ماڵی سوپەریوزەر |
| `/sbin` | فەرمانی سیستەم |
| `/tmp` | فایلی کاتی |
| `/usr` | بەرنامە و داتای بەکارهێنەران |
| `/usr/bin` | فەرمانی بەکارهێنەران |
| `/usr/lib` | کتێبخانەی `/usr/bin` |
| `/usr/local` | نرمەواڵەی جێگیرکراو لەلایەن سیستەم |
| `/var` | داتای گۆڕاو (لاگ، ئیمەیل) |
| `/var/log` | فایلی تۆمارکردنی سیستەم |

---

## لینکی سیمبۆلی
### Symbolic Links

```bash
ls -l /lib
```

```
lrwxrwxrwx  1 root root    7 2025-01-01 /lib -> usr/lib
```

`l` لە سەرەتا = **لینکی سیمبۆلی** (symbolic link). ئەمە کۆرتکوتییەک بۆ پووشپەری یان فایلی تر.
