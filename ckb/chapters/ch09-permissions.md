# بابەتی ٩ — مۆڵەتەکان
### Chapter 9 – Permissions

---

سیستەمی کارپێکردنی یونیکس-مانەند نەتەنها **زۆرکارییانە** (multitasking) بەڵکو **زۆربەکارهێنەرانەیش** (multi-user) ن. چەند بەکارهێنەر دەتوانن لەکاتیکدا لە یەک کۆمپیوتەر کار بکەن — بۆ نموونە لە ڕێگەی `ssh`. ئەم تواناکارییە پێویستیی مەکانیزمێکی پارێزگاری لە بەکارهێنەران دەرهەقی یەکتریان هەیە.

فەرمانەکانی ئەم بابەتە:
- `id` — نیشاندانی ناسنامەی بەکارهێنەر
- `chmod` — گۆڕینی مۆڵەتی فایل
- `umask` — دانانی مۆڵەتی بنچینەیی
- `su` — جێبەجێکردنی شێل وەک بەکارهێنەری تر
- `sudo` — جێبەجێکردنی فەرمان وەک بەکارهێنەری تر
- `chown` — گۆڕینی خاوەنی فایل
- `chgrp` — گۆڕینی گروپی خاوەنی فایل
- `passwd` — گۆڕینی وشەی نهێنی

---

## بەکارهێنەران، ئەندامانی گروپ و تریش

کاتێک لە بابەتی ٣دا سیستەم گەڕاینەوە، ئەگەر ئەم هەڵەیەمان بینی:

```bash
file /etc/shadow
```

```
/etc/shadow: regular file, no read permission
```

```bash
less /etc/shadow
```

```
/etc/shadow: Permission denied
```

مۆدێلی ئەمنیەتی یونیکس:
- **بەکارهێنەر** دەتوانێت خاوەنی فایل و فۆڵدەر بێت
- **گروپ** کۆمەڵێکە لە یەک یان زیاتر بەکارهێنەر کە خاوەن دەتوانێت دەستڕاگەیشتن پێی بدات
- **تریش** (others / world) بقیەی بەکارهێنەران

بۆ بینینی ناسنامەمان:

```bash
id
```

```
uid=500(me) gid=500(me) groups=500(me)
```

ئەم زانیاریانە لە فایلی دەقییەکان دێت:
- `/etc/passwd` — ئەژمێری بەکارهێنەران
- `/etc/group` — گروپەکان
- `/etc/shadow` — وشەی نهێنییە ر(کۆدکراو)

## خوێندنەوە، نووسین و جێبەجێکردن
### Reading, Writing, and Executing

```bash
ls -l foo.txt
```

```
-rw-rw-r-- 1 me me 0 2016-03-06 14:52 foo.txt
```

١٠ پیتی یەکەم **تایبەتمەندییەکانی فایل** ن:

- پیتی یەکەم: جۆری فایل
  - `-` فایلی ئاسایی
  - `d` فۆڵدەر
  - `l` لینکی سیمبۆلی

- ٩ پیتی ماوە: **دۆخی فایل** (file mode)

```
خاوەن    گروپ    تریش
rwx      rwx      rwx
```

**مانای مۆڵەتەکان:**

| مۆڵەت | فایل | فۆڵدەر |
|-------|------|--------|
| `r` | مۆڵەتی کردنەوە و خوێندنەوە | مۆڵەتی فێرستکردنی ناوەرۆک |
| `w` | مۆڵەتی نووسین | مۆڵەتی دروستکردن، سڕینەوە و ناودەستکردنەوەی فایل |
| `x` | مۆڵەتی جێبەجێکردن | مۆڵەتی چوونەژوورەوە بە `cd` |

---

## `chmod` — گۆڕینی مۆڵەت

تەنها خاوەنی فایل یان سوپەریوزەر دەتوانێت مۆڵەت بگۆڕێت.

### شێوازی ئۆکتال (Octal)

| ئۆکتال | باینەری | مۆڵەت |
|--------|---------|-------|
| 0 | 000 | `---` |
| 1 | 001 | `--x` |
| 2 | 010 | `-w-` |
| 3 | 011 | `-wx` |
| 4 | 100 | `r--` |
| 5 | 101 | `r-x` |
| 6 | 110 | `rw-` |
| 7 | 111 | `rwx` |

```bash
chmod 600 foo.txt
ls -l foo.txt
```

```
-rw------- 1 me me 0 2016-03-06 14:52 foo.txt
```

### شێوازی سیمبۆلی (Symbolic)

```
chmod [ugoa][+-=][rwx] فایل
```

| نیشان | مانا |
|-------|------|
| `u` | خاوەن (user) |
| `g` | گروپ (group) |
| `o` | تریش (others) |
| `a` | هەموو (all) |

```bash
chmod u+x file          # زیادکردنی مۆڵەتی جێبەجێکردن بۆ خاوەن
chmod u-x file          # لابردنی مۆڵەتی جێبەجێکردن لە خاوەن
chmod +x file           # زیادکردنی مۆڵەتی جێبەجێکردن بۆ هەموو
chmod go=rw file        # دانانی خوێندنەوە/نووسین بۆ گروپ و تریش
chmod u+x,go=rx file    # چەند گۆڕانکاری بە کۆما جیادەکرێن
```

## `umask` — مۆڵەتی بنچینەیی

```bash
umask
```

```
0002
```

`umask` ماسکێکە کە بیتەکان لە مۆڵەت لادەبات. ئەگەر مۆڵەتی بنچینەیی `rw-rw-rw-` بێت و umask 0002 بێت، نووسینی «تریش» لادەبرێت = `rw-rw-r--`.

---

## مۆڵەتی تایبەت

- **setuid** (ئۆکتال 4000): بەرنامە بە مۆڵەتی خاوەنەکەی جێبەجێدەکرێت. `chmod u+s program`
- **setgid** (ئۆکتال 2000): فایلی نوێ لە فۆڵدەردا گروپی فۆڵدەر مێراث دەبات. `chmod g+s dir`
- **sticky bit** (ئۆکتال 1000): تەنها خاوەن یان سوپەریوزەر دەتوانێت فایل بسڕێتەوە (وەک `/tmp`). `chmod +t dir`

## گۆڕینی ناسنامە

### `su` — جێبەجێکردنی شێل وەک بەکارهێنەری تر

```bash
su -
```

```
Password:
[root@linuxbox ~]#
```

بە `exit` دەگەڕێینەوە. بۆ جێبەجێکردنی یەک فەرمان:

```bash
su -c 'ls -l /root/*'
```

### `sudo` — جێبەجێکردنی فەرمان وەک بەکارهێنەری تر

`sudo` تواناکانی زیاتری هەیە:
- بەڕێوەبەر دەتوانێت مۆڵەتی دقیق پێناسە بکات
- پێویستی بە وشەی نهێنی سوپەریوزەر نییە
- شێلی نوێ نادەکرێتەوە

```bash
sudo backup_script
sudo -l                 # بینینی مۆڵەتە بەخشراوەکان
```

## `chown` — گۆڕینی خاوەن

```bash
chown [خاوەن][:[گروپ]] فایل...
```

| ئارگومان | ئەنجام |
|---------|-------|
| `bob` | خاوەنی گواستراوە بۆ bob |
| `bob:users` | خاوەن بۆ bob و گروپ بۆ users |
| `:admins` | تەنها گروپ بۆ admins |
| `bob:` | خاوەن بۆ bob، گروپ بۆ گروپی چوونەژوورەوەی bob |

---

## `chgrp` — گۆڕینی گروپ

مانەندی `chown` بەڵام تەنها گروپ دەگۆڕێت.

---

## تمرینی کاری: فۆڵدەری هاوبەش

janet و tony دەیانەوێت فۆڵدەری مووسیقای هاوبەش هەبێت:

```bash
sudo groupadd music
sudo usermod -a -G music janet
sudo usermod -a -G music tony
sudo mkdir /usr/local/share/Music
sudo chown :music /usr/local/share/Music
sudo chmod 2775 /usr/local/share/Music
ls -ld /usr/local/share/Music
```

```
drwxrwsr-x 2 root music 4096 /usr/local/share/Music
```

بۆ ئەوەی فایلی دروستکراو لەلایەن هەردووکیان دەستکاریکرابێت:

```bash
umask 0002
```
