# بابەتی ٨ — ئامرازی پەرەپێدراوی کیبۆرد
### Chapter 8 – Advanced Keyboard Tricks

---

زۆری دەڵێم یونیکس «سیستەمی کارپێکردنێکە بۆ ئەوانەی خۆیان تایپکردن باشەکەیان». بەڵام بەکارهێنەری ڕاستەقینەی خێزانی فەرمان بەراستی **نەیانەوێت** زۆر تایپ بکەن! بۆچی فەرمانەکان ناوی کورتیان هەیە وەک `cp`، `ls`، `mv` و `rm`؟ دەزانی چرا. یەکێک لە ئامانجە سەرەکییەکانی خێزانی فەرمان: **کەمترین تایپ بکە**. فەرمانەکانی نوێ:

- `clear` — پاккردنەوەی شاشە
- `history` — نیشاندانی مێژووی فەرمانەکان

---

## دەستکاریکردنی خێزانی فەرمان
### Command Line Editing

bash کتێبخانەیەکی بە ناوی **Readline** بۆ دەستکاریکردنی خێزانی فەرمان بەکاردێنێت.

### جووڵانی کەرسەر

| کلیل | کارکردن |
|------|--------|
| `Ctrl-a` | کۆچ بۆ سەرەتای هێڵ |
| `Ctrl-e` | کۆچ بۆ کۆتایی هێڵ |
| `Ctrl-f` | یەک پیت پێشوو |
| `Ctrl-b` | یەک پیت دواوە |
| `Alt-f` | یەک وشە پێشوو |
| `Alt-b` | یەک وشە دواوە |
| `Ctrl-l` | پاككردنەوەی شاشە |

### دەستکاریکردنی دەق

| کلیل | کارکردن |
|------|--------|
| `Ctrl-d` | سڕینەوەی پیتی ژێر کەرسەر |
| `Ctrl-t` | جووڵاندنی پیتی ژێر کەرسەر لەگەڵ پێشوو |
| `Alt-t` | جووڵاندنی وشەی ژێر کەرسەر لەگەڵ پێشوو |
| `Alt-l` | گۆڕینی لە کەرسەر تا کۆتایی وشە بۆ پیتی بچووک |
| `Alt-u` | گۆڕینی لە کەرسەر تا کۆتایی وشە بۆ پیتی گەورە |

### بڕکردن و دانانەوە (Cut & Paste)

لە مستەنداتی Readline، بە جێی «cut» `kill` و بە جێی «paste» `yank` بەکاردێت. بڕاوەکان لە بیری بە ناوی **kill-ring** دەپارێزرێن.

| کلیل | کارکردن |
|------|--------|
| `Ctrl-k` | بڕینی لە کەرسەر تا کۆتایی هێڵ |
| `Ctrl-u` | بڕینی لە کەرسەر تا سەرەتای هێڵ |
| `Alt-d` | بڕینی لە کەرسەر تا کۆتایی وشە |
| `Alt-Backspace` | بڕینی لە کەرسەر تا سەرەتای وشە |
| `Ctrl-y` | دانانەوە لە kill-ring |

## تەواوکردنی خۆکار (Completion)

کاتی تایپکردن، Tab بفشار:

```bash
ls l
```

(Tab بفشار)

```bash
ls ls-output.txt
```

ئەگەر چەند ئایتمێک هەبوو، Tab دووجار بفشار تا گەزینەکان ببینیت.

تەواوکردنی خۆکار بۆ ئەمانەیش کار دەکات:
- **گۆڕاوەکان** (ئەگەر وشە بە `$` دەستیپێدەکات)
- **ناوی بەکارهێنەران** (ئەگەر بە `~` دەستیپێدەکات)
- **فەرمانەکان** (ئەگەر یەکەم وشەی هێڵ بوو)
- **ناوی میزبان** (ئەگەر بە `@` دەستیپێدەکات)

| کلیل | کارکردن |
|------|--------|
| `Alt-?` | نیشاندانی فێرستی تەواوکردنی بەردەست |
| `Alt-*` | داخلکردنی هەموو تەواوکردنی بەردەست |

## بەکارهێنانی مێژوو
### Using History

bash مێژووی فەرمانەکانی لە فایلی `~/.bash_history` دەپارێزێت. بە بنچینە دواین **١٠٠٠** فەرمان پارێزرێت.

بۆ بینینی مێژوو:

```bash
history | less
```

### گەڕان لە مێژوودا

```bash
history | grep /usr/bin
```

```
88  ls -l /usr/bin > ls-output.txt
```

بۆ جێبەجێکردنی بە ژمارە:

```bash
!88
```

**گەڕانی پێشرو بەپێچەوانە:** **Ctrl-r** بفشار:

```
(reverse-i-search)`':
```

دەستی تایپکردن بکە:

```
(reverse-i-search)`/usr/bin': ls -l /usr/bin > ls-output.txt
```

Enter بۆ جێبەجێکردن، یان **Ctrl-j** بۆ کۆپیکردنی بۆ خێزانی ئێستا.

**کلیلەکانی بەسوود بۆ مێژوو:**

| کلیل | کارکردن |
|------|--------|
| `Ctrl-p` | فەرمانی پێشوو |
| `Ctrl-n` | فەرمانی داهاتوو |
| `Alt-<` | کۆچ بۆ یەکەم فەرمانی مێژوو |
| `Alt->` | کۆچ بۆ کۆتایی مێژوو |
| `Ctrl-r` | گەڕانی پێشروی بەپێچەوانە |
| `Ctrl-o` | جێبەجێکردن و کۆچ بۆ داهاتوو |

### فراوانکردنی مێژوو (History Expansion)

| زنجیرە | کارکردن |
|--------|--------|
| `!!` | دووبارەکردنەوەی فەرمانی دوایین |
| `!ژمارە` | دووبارەکردنەوەی فەرمانی ژمارەی دیاری |
| `!زنجیرە` | دووبارەکردنی دواین فەرمان لەگەڵ ئەو زنجیرەیەوە |
| `!?زنجیرە` | دووبارەکردنی دواین فەرمان کە ئەو زنجیرەی تێدایە |

## script — تۆمارکردنی نیشستی تێرمینال

زۆری توزیعەکانی لینوکس فەرمانێکی بە ناوی `script` هەیانە کە هەموو نیشستی تێرمینال تۆمار دەکات:

```bash
script [ناوی_فایل]
```

ئەگەر ناوی فایل نەدرا، بە بنچینە `typescript` بەکاردێت.
