## کۆمەڵکردنی فەرمان (Command Grouping)

دوو ڕێگا بۆ کۆمەڵکردنی فەرمانەکان:

### ١. ژێرشێل `( )`

```bash
(cd /tmp; ls)
```

فەرمانەکان لە **ژێرشێلێکی** (subshell) جیاوازدا جێبەجێدەبن. گۆڕانی گۆڕاو و پووشپەری ئێستا کاریگەری لەسەر شێلی سەرەکی نییە.

### ٢. کۆمەڵکەر `{ }`

```bash
{ cd /tmp; ls; }
```

فەرمانەکان لە **شێلی ئێستادا** جێبەجێدەبن. گۆڕانکارییەکان دەمێننەوە.

### نموونەی کاربەکار

```bash
{ echo "Header"; cat file.txt; echo "Footer"; } > output.txt
```

## جێگرینی پرۆسەس (Process Substitution)

```bash
command <(list)
command >(list)
```

ئاوتپوتی فەرمان وەک فایلێک لە بەرچاو دەگیرێت:

```bash
diff <(ls dir1) <(ls dir2)
```

فایلەکانی دوو پووشپەر بەراوردئەکاتەوە — بەبێ پێویستی بە فایلی کاتی.

---

## `trap` — گرتنی سیگنال

```bash
trap argument signal...
```

```bash
#!/bin/bash

cleanup () {
    echo "Cleaning up..."
    rm -f /tmp/mytemp*
    exit
}

trap cleanup EXIT INT TERM

tmpfile=$(mktemp /tmp/mytemp.XXXXXX)
echo "Working with $tmpfile"
sleep 30
```

### سیگناڵەکانی باو

| سیگناڵ | مانا |
|--------|------|
| `EXIT` | دەرچوون لە ئیسکریپت |
| `INT` | Ctrl+C |
| `TERM` | داواکاری کۆتایی |
| `KILL` | کۆتایی بە زۆر |
| `HUP` | بڕینی پەیوەندی |

## `mapfile` — خوێندنەوەی فایل بۆ ئەرەی

```bash
mapfile -t lines < /etc/passwd
echo "Lines: ${#lines[@]}"
echo "First: ${lines[0]}"
```

## جێگرینی ناوی فایلی پێشکەوتوو

### `**` — globbing بازگشتی

```bash
shopt -s globstar
echo **/*.txt       # هەموو فایلەکانی .txt لە هەموو ژێرپووشپەرەکان
```

### `extglob` — شێوازی پێشکەوتوو

```bash
shopt -s extglob

ls !(*.jpg)         # هەموو شت بجگە لە .jpg
ls +(foo|bar)       # یەک یان زیاتر لە foo یان bar
ls ?(foo|bar)       # سفر یان یەک لە foo یان bar
```

## ماژولکردنی ئیسکریپت بە `source`

```bash
# lib.sh
greet () {
    echo "Hello, $1!"
}

# main.sh
source ./lib.sh
greet "World"
```

یان بە خاڵ:

```bash
. ./lib.sh
```
