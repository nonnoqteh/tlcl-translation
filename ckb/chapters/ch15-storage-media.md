# بابەتی ١٥ — میدیای ستۆرکردن
### Chapter 15 – Storage Media

---

فەرمانەکانی ئەم بابەتە:

- `mount` — سواربوونی سیستەمی فایل
- `umount` — داپیاوبوونی سیستەمی فایل
- `fsck` — پشکنین و چاکردنەوەی سیستەمی فایل
- `fdisk` — دەستکاریکردنی جەدوەلی پارتیشن
- `mkfs` — دروستکردنی سیستەمی فایل
- `dd` — گواستنەوە و کۆپیکردنی فایل
- `genisoimage` — دروستکردنی فایلی وێنەی ISO
- `wodim` — نووسین لەسەر میدیای نووری
- `md5sum` — ژماردنی چێکسەمی MD5

---

## بەڕێوەبردنی ئامێرەکانی ستۆرکردن

### سواربوون و داپیاوبوون

لە لینوکس، ئامێرەکانی ستۆرکردن بۆ درەختی سیستەمی فایل **سوار** (mount) دەبن.

بۆ بینینی ئامێرە سوارکراوەکان:

```bash
mount
```

```
/dev/sda1 on / type ext4 (rw,relatime)
/dev/sdb1 on /media/disk type vfat (rw,nosuid,nodev)
```

#### سواربوونی فلاش USB یان CD

```bash
mkdir /mnt/usb
mount /dev/sdb1 /mnt/usb
ls /mnt/usb
```

#### داپیاوبوون

```bash
umount /mnt/usb
```

> **گرنگ:** پێش جیاکردنەوەی ئامێر، هەمیشە داپیاویت بکە تا داتاکان زیانی نەبینن.

---

## پشکنینی سیستەمی فایل

```bash
sudo fsck /dev/sdb1
```

> **ئاگاداری:** هیچکات `fsck` لەسەر سیستەمی فایلی سوارکراو مەجێبەجێبکە!

---

## پارتیشنبەندی

```bash
sudo fdisk /dev/sdb
```

فەرمانی گرنگ لە `fdisk`:

| کلیل | کارکردن |
|------|--------|
| `p` | نیشاندانی جەدوەلی پارتیشن |
| `n` | دروستکردنی پارتیشنی نوێ |
| `d` | سڕینەوەی پارتیشن |
| `t` | گۆڕینی جۆری پارتیشن |
| `w` | نووسینی گۆڕانکاری و دەرچوون |
| `q` | دەرچوون بەبێ پاراستن |
| `m` | نیشاندانی یارمەتی |

---

## دروستکردنی سیستەمی فایلی نوێ

دوای پارتیشنبەندی، پێویستە سیستەمی فایل بدروستبکەین:

```bash
# دروستکردنی ext4
sudo mkfs -t ext4 /dev/sdb1

# دروستکردنی FAT32 (بۆ گونجاندن لەگەڵ ویندۆز)
sudo mkfs -t vfat /dev/sdb1
```

---

## دستووری `dd` — کۆپیکردنی ئاستی نزم

`dd` بلۆک بە بلۆک داتا کۆپیدەکات:

```bash
dd if=input_file of=output_file [bs=block_size]
```

| پارامێتەر | مانا |
|----------|------|
| `if=` | فایلی ئینپوت |
| `of=` | فایلی ئاوتپوت |
| `bs=` | قەبارەی بلۆک |

**نموونەکان:**

```bash
# کۆپیکردنی تەواوی درایڤێک بۆ درایڤی تر
dd if=/dev/sda of=/dev/sdb

# دروستکردنی فایلی وێنە لە فلاش USB
dd if=/dev/sdb of=usb_backup.img

# نووسینی ISO لەسەر فلاش USB
dd if=ubuntu.iso of=/dev/sdb bs=4M status=progress
```

> ⚠️ **ئاگاداری:** `dd` بەبێ ئاگادارکردنەوە داتا دەنووسێتەوە. دڵنیا بە لە `if` و `of`!

---

## کار لەگەڵ CD/DVD

### دروستکردنی ISO لە CD

```bash
dd if=/dev/cdrom of=disc_image.iso
```

### سووتاندن لەسەر CD/DVD

```bash
wodim dev=/dev/cdrw image.iso
```

### پشتیراستکردن بە MD5

```bash
md5sum image.iso
```
