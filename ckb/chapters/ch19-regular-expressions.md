# بابەتی ١٩ — دەقی ئایەتی (Regular Expressions)
### Chapter 19 – Regular Expressions

---

فەرمانەکانی ئەم بابەتە:

- `grep` — گەڕان بۆ نموونە لە دەق
- `sed` — ئامرازی دەستکاریکردنی دەق
- `awk` — زمانی پرۆگرامنووسی بۆ دەق

---

## دەقی ئایەتی چییە؟

**دەقی ئایەتی** (regular expression / regex) زمانێکی تایبەتە بۆ وەسفکردنی نموونەی دەق. بە کورتی: رێگایەکی کودکراوی بۆ گەڕان و دۆزینەوەی نموونە.

```bash
grep -h '.zip' dirlist*.txt
```

---

## مێتاکاراکتەرەکان

دەقی ئایەتی دوو جۆر کاراکتەر هەیە:
- **ئایەتی** (literal): کاراکتەرە ئاسایییەکان
- **مێتاکاراکتەر**: کاراکتەرە تایبەتەکان — `. ^ $ * + ? { } [ ] \ | ( )`

### `.` — هەر کاراکتەرێک

```bash
grep -h '.zip' dirlist*.txt
```

`.` لە شوێنی هەر کاراکتەرێک دەوەستێت.

---

## کاراکتەری چین `[ ]`

### دیاریکردنی چین

```bash
grep -h '[bg]zip' dirlist*.txt
```

دەگەڕێت بۆ `bzip` یان `gzip`.

### نێگەشن (Negation)

```bash
grep -h '[^bg]zip' dirlist*.txt
```

دەگەڕێت بۆ `zip` کە پێشەکەی نە `b` نە `g` نییە.

### ئارەستەکان (Ranges)

```bash
grep -h '[a-z]' dirlist*.txt    # هەر پیتی بچووک
grep -h '[A-Z]' dirlist*.txt    # هەر پیتی گەورە
grep -h '[0-9]' dirlist*.txt    # هەر ژمارەیەک
grep -h '[a-zA-Z0-9]' dirlist*.txt   # هەموو
```

### چینە پێش پێناسەکراوەکان

| چین | مانا |
|-----|------|
| `[:alnum:]` | پیت و ژمارە `[A-Za-z0-9]` |
| `[:alpha:]` | پیتەکان `[A-Za-z]` |
| `[:digit:]` | ژمارەکان `[0-9]` |
| `[:lower:]` | پیتی بچووک `[a-z]` |
| `[:upper:]` | پیتی گەورە `[A-Z]` |
| `[:space:]` | فاسیلە، tab، newline |
| `[:punct:]` | نیشانەکانی نووستن |

```bash
grep -h '[[:upper:]]' dirlist*.txt
```

---

## لانگەرەکان (Anchors)

```bash
grep -h '^zip' dirlist*.txt     # dest زیاتر نابینێ
grep -h 'zip$' dirlist*.txt     # کۆتا بوو
grep -h '^zip$' dirlist*.txt    # تەنها "zip"
```

---

## دەستکاریکردن بە `grep`

```bash
grep -i 'pattern' file          # بەبێ ئاگاداری گەورە/بچووک
grep -v 'pattern' file          # هێڵەکانی کە نموونە ناگرن
grep -c 'pattern' file          # ژمارەی هێڵەکانی گونجاو
grep -n 'pattern' file          # ژمارەی هێڵ لەگەڵ
grep -l 'pattern' *.txt         # تەنها ناوی فایلەکان
grep -E 'pattern' file          # دەقی ئایەتی پێشکەوتوو
```

---

## دەقی ئایەتی پێشکەوتوو (ERE)

بە `-E` یان `egrep`:

### بەدووانان `|`

```bash
grep -E 'AAA|BBB|CCC' file
```

### کاراکتەرەکانی دووبارە

| نیشانە | مانا |
|--------|------|
| `?` | سفر یان یەک جار |
| `*` | سفر یان زیاتر |
| `+` | یەک یان زیاتر |
| `{n}` | تەواو n جار |
| `{n,}` | n جار یان زیاتر |
| `{n,m}` | لە n تا m |

```bash
grep -E '^\(?[0-9]{3}\)? [0-9]{3}-[0-9]{4}$' file   # ژمارەی تەلەفۆن
```

### گروپکردن `( )`

```bash
grep -E '^(bz|gz|zip)' dirlist*.txt
```

---

## `sed` — دەستکاریکردنی دەقی دەورەیی

`sed` دەقی ئینپوت بۆ ئاوتپوت گواستنەوە و نموونە جێگردەکاتەوە.

### جێگرینەوە

```bash
sed 's/old/new/' file          # یەک جار هەر هێڵ
sed 's/old/new/g' file         # هەمووی
sed -i 's/old/new/g' file      # نووسینەوەی ڕاستەوخۆ
```

### نموونەی تایبەت

```bash
# سڕینەوەی هێڵە خاڵی
sed '/^$/d' file

# نیشاندانی هێڵی ١ تا ٥
sed -n '1,5p' file

# سڕینەوەی هێڵێکی تایبەت
sed '5d' file
```

---

## کورتە

دەقی ئایەتی یەکێک لە بەهێزترین ئامرازەکانی پرۆگرامی لینوکسە. `grep` بۆ گەڕان، `sed` بۆ جێگرینەوە. تەمرین کردنیان توانای ئامرازەکانی دەقی لینوکس زۆر بەرزدەکاتەوە.
