# بابەتی ١٣ — دیزاینکردنی پرامپت
### Chapter 13 – Customizing the Prompt

---

لەم بابەتەدا بە بابەتێکی کێفخۆش دەپردازین: **دیزاینکردنی پرامپت** (prompt customization). پرامپت ئەو زنجیرەیەیە کە هەر جار پێش داخلکردنی فەرمانێک دەردەکەوێت — مانەندی `[me@linuxbox ~]$`.

---

## پرامپت چۆن کار دەکات؟
### Anatomy of a Prompt

پرامپت لەلایەن گۆڕاوی ژینگەی `PS1` پێناسە دەکرێت (کورتی «Prompt String 1»). ناوەرۆکەکەی بخوێنینەوە:

```bash
echo $PS1
```

```
[\u@\h \W]\$
```

ئەم زنجیرەیە **زنجیرەی فیراری** (escape sequences) تێدایە کە bash جێگرینیان دەکات:

| زنجیرە | جێگرین |
|--------|--------|
| `\a` | دەنگی زەنگ |
| `\d` | بەروار بە فۆرماتی «Weekday Month Date» |
| `\h` | ناوی میزبان تا یەکەم خاڵ |
| `\H` | ناوی میزبانی تەواو |
| `\j` | ژمارەی کارەکانی چالاک |
| `\l` | ناوی ئامێری تێرمینال |
| `\n` | هێڵی نوێ |
| `\s` | ناوی شێل |
| `\t` | کات بە فۆرماتی ٢٤ کاتژمێرە (HH:MM:SS) |
| `\T` | کات بە فۆرماتی ١٢ کاتژمێرە |
| `\@` | کات بە AM/PM |
| `\u` | ناوی بەکارهێنەر |
| `\v` | وەشانی bash |
| `\V` | وەشانی تەواوی bash |
| `\w` | ڕێچکەی پووشپەری کاری ئێستا |
| `\W` | ناوی پووشپەری کاری ئێستا (بەشی کۆتایی) |
| `\!` | ژمارەی مێژووی ئەم فەرمانە |
| `\#` | ژمارەی فەرمانی ئێستا |
| `\$` | `$` بۆ بەکارهێنەری ئاسایی، `#` بۆ root |
| `\\` | بەکسلاش |
| `\[` | دەستپێکی زنجیرەی نەچاپراو |
| `\]` | کۆتایی زنجیرەی نەچاپراو |

---

## دروستکردنی پرامپتی نوێ

پرامپتێکی سادە بە زانیاری زیاتر:

```bash
PS1="\u@\h:\w\$ "
```

ئەنجام:

```
me@linuxbox:~$ 
```

یان بە نیشاندانی کات:

```bash
PS1="\A \u@\h:\w\$ "
```

```
17:33 me@linuxbox:~$ 
```

## زیادکردنی ڕەنگ
### Adding Color

تێرمینالی مۆدێرن لە **کۆدەکانی فیراری ANSI** بۆ ڕەنگ پشتیبانی دەکات:

```
\[\033[کۆدی‌ڕەنگ\]
```

**ڕەنگەکانی دەق:**

| کۆد | ڕەنگ |
|-----|------|
| `0;30` | ڕەش |
| `0;31` | سوور |
| `0;32` | سەوز |
| `0;33` | قاوەیی/زەرد |
| `0;34` | شین |
| `0;35` | مۆر |
| `0;36` | فیرۆزەیی |
| `0;37` | سپی |
| `1;31` | سووری ڕووناک |
| `1;32` | سەوزی ڕووناک |
| `1;33` | زەرد |
| `1;34` | شینی ڕووناک |
| `1;35` | مۆری ڕووناک |
| `1;36` | فیرۆزەیی ڕووناک |
| `1;37` | سپی |

**بۆ ڕیستکردنەوەی ڕەنگ:** `\[\033[0m\]`

**نموونەی کارامە — پرامپتی سەوز:**

```bash
PS1="\[\033[1;32m\]\u@\h:\w\$\[\033[0m\] "
```

**نموونە بە ناوی بەکارهێنەری شین و ڕێچکەی سەوز:**

```bash
PS1="\[\033[1;34m\]\u@\h\[\033[0m\]:\[\033[1;32m\]\w\[\033[0m\]\$ "
```

**پرامپتی root بە ڕەنگی سوور** (ئاگادارکردنەوەی بینایی):

```bash
PS1="\[\033[1;31m\][\u@\h \W]\$\[\033[0m\] "
```

### Background Colors

کۆدەکانی پشتزەوی بە `4` دەستیدەکەن:

| `43` | قاوەیی |
| `44` | شین |
| `45` | مۆر |

## پاراستنی پرامپت لە ڕێکخستنەکان

دوای دۆزینەوەی پرامپتی ئیده‌آل، لە `~/.bashrc` زیادی بکە:

```bash
# پرامپتی تایبەت
PS1="\[\033[1;32m\]\u@\h\[\033[0m\]:\[\033[1;34m\]\w\[\033[0m\]\$ "
export PS1
```

دوای پاراستن:

```bash
source ~/.bashrc
```

---

## پرامپتی تریش

سەرەڕای `PS1`، گۆڕاوی پرامپتی تری هەیە:

| گۆڕاو | بەکارهێنان |
|-------|-----------|
| `PS1` | پرامپتی سەرەکی |
| `PS2` | پرامپتی دووەم (کاتێک فەرمان بەردەوامە) |
| `PS3` | پرامپت بۆ فەرمانی `select` |
| `PS4` | پرامپت لە کاتی هەڵەیابینی |

نموونەی پرامپتی دووەم:

```bash
$ echo "چەند
> هێڵ"
```

`>` هەمان `PS2` ە.

## کورتە

دیزاینکردنی پرامپت یەکێک لە شێوازە کێفخۆشەکانی تایبەتیکردنی ئەزموونی لینوکسە. پرامپتی باش زانیاری بەسوود نیشاندەدات و لەکاتیکدا شلووپ نیە. تاقی بکەرەوە و پرامپتێک بدروست بکە کە بۆت گونجاو و کاریگەرە.
