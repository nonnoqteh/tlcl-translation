# بابەتی ١٢ — ناساندنی vi
### Chapter 12 – A Gentle Introduction to vi

---

کاتێک یەکەم جار لەگەڵ لینوکس کار کردم، زوو تێگەیشتم کە زانیاری دەستکاری‌کەری دەق تواناکارییەکی ژیانی‌یە. مانەندی زۆری تواناکارییەکانی لینوکس، فێربوونی لە سەرەتا سەختی دەربینێت بەڵام دوای تسەلیم، نایابانە هێزدار دەبێت.

فەرمانی ئەم بابەتە:
- `vi` — دەستکاری‌کەری دەق

---

## بۆچی vi فێربین؟
### Why We Should Learn vi

لە دنیای لینوکس، دەستکاری‌کەری گرافیکی زۆرن مانەندی gedit و Kate... بەڵام vi (یان vim — وەشانی باشترکراوی) هەندێک هۆکاری گرنگی هەیە بۆ فێربوون:

- **هەمیشە و هەموو شوێنەوە هەیە:** vi لە هەموو سیستەمی یونیکس-مانەند نصبکراوە — لە سێرڤەری بچووکەوە تا سوپەرکۆمپیوتەر. ئەگەر لەسەر سێرڤەرێکی بەبێ ڕووکاری گرافیکی کار دەکەیت، vi تەنها هەڵبژاردنی بەردەستە.
- **سووکە:** لەسەر لاوازترین سیستەمیشدا کار دەکات.
- **دوای تسەلیم نایابانە خێراتر:** بەکارهێنەری پیشەیی بەبێ هەڵگرتنی دەست لە کیبۆرد کار دەکەن.

---

## دەستپێکردنی کار لەگەڵ vi

```bash
vi
```

یان بۆ کردنەوەی فایل:

```bash
vi foo.txt
```

### حاڵەتەکانی vi
### vi Modes

**گرنگترین شت کە پێویستە بزانیت:** vi **زۆرحاڵەتییە** (multimodal). دوو حاڵەتی سەرەکی هەیە:

- **حاڵەتی فەرمان** (Command mode) — حاڵەتی بنچینەیی. لەم حاڵەتەدا کلیلەکان فەرمانن، نەک دەق.
- **حاڵەتی داخلکردن** (Insert mode) — لەم حاڵەتەدا دەتوانین دەق تایپ بکەین.

> **گەورەترین کێشەی سەرەتاییەکان:** لەبیرکردنی لە کام حاڵەتن! هەر جار گیر کەوتی، `Esc` بفشار بۆ گەڕانەوە بۆ حاڵەتی فەرمان.

---

## جووڵان لە vi دا

لە حاڵەتی فەرمان، ئەم کلیلانە کەرسەر جووڵاندەن:

| کلیل | جووڵان |
|------|--------|
| `l` یان `→` | یەک پیت بەرەو ڕاست |
| `h` یان `←` | یەک پیت بەرەو چەپ |
| `j` یان `↓` | یەک هێڵ خوارەوە |
| `k` یان `↑` | یەک هێڵ سەرەوە |
| `0` | کۆچ بۆ سەرەتای هێڵ |
| `^` | کۆچ بۆ یەکەم پیتی نافاسیلەی هێڵ |
| `$` | کۆچ بۆ کۆتایی هێڵ |
| `w` | کۆچ بۆ سەرەتای وشەی داهاتوو |
| `b` | کۆچ بۆ سەرەتای وشەی پێشوو |
| `G` | کۆچ بۆ کۆتایین هێڵی فایل |
| `1G` یان `gg` | کۆچ بۆ یەکەم هێڵی فایل |
| `nG` | کۆچ بۆ هێڵی n |
| `Ctrl-f` | یەک لاپەڕە خوارەوە |
| `Ctrl-b` | یەک لاپەڕە سەرەوە |

---

## داخلکردنی دەق
### Inserting Text

لە حاڵەتی فەرمان بۆ چوونە حاڵەتی داخلکردن:

| کلیل | کارکردن |
|------|--------|
| `i` | داخلکردن پێش پیتی ئێستا |
| `a` | داخلکردن دوای پیتی ئێستا |
| `A` | داخلکردن لە کۆتایی هێڵ |
| `o` | کردنەوەی هێڵی نوێ لە ژێر هێڵی ئێستا |
| `O` | کردنەوەی هێڵی نوێ لە سەرەوەی هێڵی ئێستا |

دوای تایپکردنی دەق، بە `Esc` بگەڕێوە بۆ حاڵەتی فەرمان.

---

## پاراستن و دەرچوون
### Saving Our Work

لە vi فەرمانەکانی **ex** (بە `:`) بۆ پاراستن و دەرچوون بەکاردێن:

| فەرمان | کارکردن |
|--------|--------|
| `:w` | پاراستنی فایل |
| `:w filename` | پاراستن بە ناوی نوێ |
| `:wq` | پاراستن و دەرچوون |
| `:q` | دەرچوون (تەنها ئەگەر گۆڕانکاری نەکراوبێت) |
| `:q!` | دەرچوونی زەبری بەبێ پاراستن |
| `ZZ` | پاراستن و دەرچوون (بەبێ دووخاڵ) |

---

## سڕینەوەی دەق
### Deleting Text

| کلیل | کارکردن |
|------|--------|
| `x` | سڕینەوەی پیتی ئێستا |
| `dd` | سڕینەوەی هێڵی ئێستا |
| `ndd` | سڕینەوەی n هێڵ |
| `dw` | سڕینەوە لە کەرسەر تا سەرەتای وشەی داهاتوو |
| `d$` | سڕینەوە لە کەرسەر تا کۆتایی هێڵ |
| `d0` | سڕینەوە لە کەرسەر تا سەرەتای هێڵ |

---

## بڕین، کۆپیکردن و دانانەوە
### Cut, Copy and Paste

لە vi بە جێی «cut» لە «yank» (کەندن) و بە جێی «copy» لە «put» بەکاردێت:

| کلیل | کارکردن |
|------|--------|
| `yy` | کۆپیکردنی هێڵی ئێستا |
| `nyy` | کۆپیکردنی n هێڵ |
| `yw` | کۆپی لە کەرسەر تا سەرەتای وشەی داهاتوو |
| `y$` | کۆپی لە کەرسەر تا کۆتایی هێڵ |
| `p` | دانانەوە دوای هێڵی ئێستا |
| `P` | دانانەوە پێش هێڵی ئێستا |

> **تێبینی:** `dd` **بڕین** ە — دەقی سڕاوە لە بافەر دەپارێزرێت و بە `p` جێنشیندەبێت.

## گەڕان و جێگرینەوە
### Search and Replace

### گەڕان

لە حاڵەتی فەرمان:

```
/وشە
```

Enter بفشار. بۆ نتیجەی داهاتوو: `n` — بۆ پێشوو: `N`.

بۆ گەڕانی بەپێچەوانە (بەرەو سەرەوە):

```
?وشە
```

### جێگرینەوەی گشتی

```
:%s/کۆن/نوێ/g
```

- `%` = لە تەواوی فایل
- `s` = جێگرینەوە (substitute)
- `g` = هەموو دووبارەکردنەوەکان لە هەر هێڵ (global)

بۆ پشتیراستکردنی هەر حاڵ پێش جێگرینەوە:

```
:%s/کۆن/نوێ/gc
```

بە `c`، vi بۆ هەر حاڵ دەپرسێت:
- `y` — جێگرین بکە
- `n` — رەتبکەرەوە
- `a` — هەموویان جێگرین بکە
- `q` — دەرچوون لە جێگرینەوە

## دەستکاریکردنی چەند فایل
### Editing Multiple Files

```bash
vi file1 file2 file3
```

| فەرمان | کارکردن |
|--------|--------|
| `:n` | کۆچ بۆ فایلی داهاتوو |
| `:N` | کۆچ بۆ فایلی پێشوو |
| `:buffers` | نیشاندانی فێرستی فایلە کراوەکان |

## ڕێکخستنەکانی بەسوودی vi

دەتوانین ڕێکخستنەکانی vi لە فەرمانی `:set` پێناسە بکەین. بۆ هەمیشەیی‌کردن، ئەم هێڵانە لە `~/.vimrc` بنووسە:

```
:set number        # نیشاندانی ژمارەی هێڵ
:set nonumber      # شاراندنی ژمارەی هێڵ
:set ignorecase    # گەڕانی بەبێ ئاگاداری گەورە/بچووک
:set hlsearch      # هایلایتکردنی ئەنجامی گەڕان
:set autoindent    # تورفەندانی خۆکار
```

## کورتە

vi لە سەرەتا گیجەڵکەرە، بەڵام دوای چەند کاتژمێر تاقیکردنەوە دەستیدەکات ئاقڵانەی بێت. گرنگترین شت: هەمیشە بزانە لە کام حاڵەتیت. لەگەڵ تاقیکردنەوەی بەردەوام، vi دەبێتە یەکێک لە هێزداترین ئامرازەکانت.
