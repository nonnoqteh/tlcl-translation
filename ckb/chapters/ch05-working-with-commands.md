# بابەتی ٥ — کارکردن لەگەڵ فەرمانەکان
### Chapter 5 – Working with Commands

---

فەرمانەکانی ئەم بابەتە:

- `type` — نیشاندانی جۆری فەرمان
- `which` — نیشاندانی شوێنی بەرنامە
- `help` — یارمەتی فەرمانی ناوخۆی شێل
- `man` — نیشاندانی لاپەڕەی راهنامای فەرمان
- `apropos` — نیشاندانی فەرمانی پەیوەندیدار
- `info` — زانیاریی تفصیلی فەرمان
- `whatis` — ڕوونکردنەوەی یەک-هێڵی
- `alias` — دروستکردنی ناوی جێگریەوە

---

## فەرمان ئاکامی چییە؟
### What Exactly Are Commands?

فەرمان دەتوانێت یەکێک لەم چوار شتانە بێت:

1. **بەرنامەی جێبەجێکراو** — وەک فایلەکانی `/usr/bin`. بەرنامەی کۆمپایڵکراو (C/C++) یان ئیسکریپت (Python, Ruby).
2. **فەرمانی ناوخۆی شێل** — bash هەندێک فەرمانی تێدا پەیدابووە. وەک `cd`.
3. **ئەمرازی شێل** — ئیسکریپتی بچووکی شێل کە لە ژینگەدا جێنشینکراوە.
4. **ناوی جێگریەوە (Alias)** — فەرمانێک کە خۆمان لە فەرمانی تر دروستدەکەین.

---

## ناسینی فەرمانەکان

### `type` — نیشاندانی جۆری فەرمان

```bash
type type
```

```
type is a shell builtin
```

```bash
type ls
```

```
ls is aliased to 'ls --color=auto'
```

```bash
type cp
```

```
cp is /usr/bin/cp
```

### `which` — نیشاندانی شوێنی بەرنامە

```bash
which ls
```

```
/usr/bin/ls
```

> `which` تەنها بۆ بەرنامەی جێبەجێکراو کار دەکات، نەک فەرمانی ناوخۆ یان ناوی جێگریەوە.

---

## مستەنداتی فەرمان

### `help` — یارمەتی فەرمانی ناوخۆ

```bash
help cd
```

```
cd: cd [-L|[-P [-e]] [-@]] [dir]
    Change the shell working directory.
    ...
```

> کاتێک `[]` دەبینیت = ئەو بەشە ئارەزوویییە. `|` = «یان».

### `--help` — نیشاندانی زانیاری بەکارهێنان

```bash
mkdir --help
```

```
Usage: mkdir [OPTION] DIRECTORY...
Create the DIRECTORY(ies), if they do not already exist.
  -m, --mode=MODE   set file mode
  -p, --parents     make parent directories as needed
  ...
```

### `man` — لاپەڕەی راهنامە

```bash
man ls
```

لاپەڕەکانی man بە `less` دەردەکەون، بۆیە هەموو کلیلەکانی `less` کار دەکەن.

**بەشەکانی راهنامە:**

| بەش | ناوەرۆک |
|-----|---------|
| ١ | فەرمانی بەکارهێنەر |
| ٢ | ڕووکاری بەرنامەنووسی هسته |
| ٣ | ڕووکاری کتێبخانەی C |
| ٤ | فایلی تایبەت (ئامێرەکان) |
| ٥ | فۆرماتی فایل |
| ٦ | یاری |
| ٧ | جۆراوجۆر |
| ٨ | فەرمانی بەڕێوەبردنی سیستەم |

```bash
man 5 passwd
```

### `apropos` — نیشاندانی پەیوەندیداران

```bash
apropos partition
```

```
addpart (8)    - simple wrapper around the "add partition"...
fdisk (8)      - manipulate disk partition table
...
```

### `whatis` — ڕوونکردنەوەی یەک-هێڵی

```bash
whatis ls
```

```
ls (1)  - list directory contents
```

### `info` — زانیاریی تفصیلی

```bash
info coreutils
```

**کلیلەکانی `info`:**

| کلیل | کارکردن |
|------|--------|
| `?` | نیشاندانی یارمەتی |
| `PgUp` | لاپەڕەی پێشوو |
| `PgDn` | لاپەڕەی داهاتوو |
| `n` | نۆدی داهاتوو |
| `p` | نۆدی پێشوو |
| `u` | نۆدی دایک |
| `Enter` | شوێنکردنی هایپەرلینک |
| `q` | دەرچوون |

## دروستکردنی فەرمانی خۆمان بە `alias`

دەتوانین چەند فەرمانێک بە `;` تێکەڵ بکەین:

```bash
cd /usr; ls; cd -
```

```
bin  games  include  lib  local  sbin  share  src
/home/me
```

ئێستا ئەمە بکەینە فەرمانی تازە. یەکەم بزانین ناوەکە بەردەستە:

```bash
type foo
```

```
bash: type: foo: not found
```

```bash
alias foo='cd /usr; ls; cd -'
foo
```

```
bin  games  include  lib  local  sbin  share  src
/home/me
```

بۆ سڕینەوەی ناوی جێگریەوە:

```bash
unalias foo
```

بۆ بینینی هەموو ناوەکانی جێگریەوە:

```bash
alias
```

> **تێبینی:** ناوی جێگریەوەی دروستکراو لە خێزانی فەرمان، کاتێک نیشست کۆتایی دێت لەدەستدەدرێن. لە بابەتی ١١دا فێردەبین چۆن بیکەینە هەمیشەیی.
