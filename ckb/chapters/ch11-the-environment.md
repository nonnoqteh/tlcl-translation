# بابەتی ١١ — ژینگە
### Chapter 11 – The Environment

---

وەک پێشتر بینیمان، شێل لە کاتی نیشست (session) زانیارییەک دەپارێزێت کە **ژینگە** (environment) ناودەبردرێت. بەرنامەکان لەم ژینگەیەوە بەکاردێن بۆ ڕیزکردنی رەفتارەکانیان. زۆری بەرنامەکان لە **فایلی ڕێکخستن** (configuration files) یش بەکاردێن.

فەرمانەکانی ئەم بابەتە:
- `printenv` — چاپکردنی بەشێک یان هەموو ژینگە
- `set` — دانانی ئۆپشنەکانی شێل
- `export` — ناردنی گۆڕاو بۆ ژینگە
- `alias` — دروستکردنی ناوی جێگریەوە بۆ فەرمان

---

## ژینگە چییە؟
### What Is Stored in the Environment?

شێل زانیارییەکان لە دوو جۆر گۆڕاودا دەپارێزێت:

- **گۆڕاوی ژینگە** (environment variables) — زانیارییەک کە لە پرۆسەی دایکەوە بۆ منداڵ مێراث دەبات
- **گۆڕاوی شێل** (shell variables) — زانیارییەک کە تەنها لە نیشستی شێلی ئێستادا دەژیت

سەرەڕای گۆڕاوەکان، شێل داتای بەرنامەنووسی مانەندی **ئەمرازەکان** (functions) و **ناوی جێگریەوەکان** (aliases) یش دەپارێزێت.

### پشکنینی ژینگە

بۆ بینینی گۆڕاوەکانی ژینگە:

```bash
printenv | less
```

بۆ بینینی گۆڕاوێکی دیاری:

```bash
printenv USER
```

```
me
```

بە `set` هەردوو جۆر گۆڕاو دەبینین:

```bash
set | less
```

بۆ بینینی ناوەرۆکی هەر گۆڕاوێک:

```bash
echo $HOME
```

```
/home/me
```

**گۆڕاوە گرنگەکانی ژینگە:**

| گۆڕاو | ناوەرۆک |
|-------|---------|
| `DISPLAY` | ناوی نیشاندەری گرافیکی |
| `EDITOR` | دەستکاری‌کەری بنچینەیی |
| `HOME` | ڕێچکەی پووشپەری ماڵ |
| `LANG` | کاراکتەرسێت و ڕیزکردن |
| `OLDPWD` | پووشپەری کاری پێشوو |
| `PAGER` | ناوی بەرنامەی لاپەڕەبەندی بنچینەیی (`less`) |
| `PATH` | فێرستی پووشپەرەکانی گەڕان بۆ جێبەجێکراوەکان |
| `PS1` | زنجیرەی پرامپتی یەکەم |
| `PWD` | پووشپەری کاری ئێستا |
| `SHELL` | ناوی شێل |
| `TERM` | جۆری تێرمینال |
| `TZ` | ناوچەی کاتی |
| `USER` | ناوی بەکارهێنەر |

---

## ژینگە چۆن دروستدەبێت؟
### How Is the Environment Established?

کاتێک چوونەژوورەوە دەکەیت، پرۆسەی bash زنجیرەیەک لە فایلی ڕێکخستن بە ناوی **فایلی دەستپێکردن** (startup files) دەخوێنێتەوە.

### نیشستی چوونەژوورەوە (Login Shell Session)

| فایل | ناوەرۆک |
|------|---------|
| `/etc/profile` | ئیسکریپتی ڕێکخستنی گشتی بۆ هەموو بەکارهێنەران |
| `~/.bash_profile` | فایلی دەستپێکردنی تاکەکەس |
| `~/.bash_login` | ئەگەر `~/.bash_profile` نەبوو، ئەمە دەخوێنرێتەوە |
| `~/.profile` | ئەگەر هیچکام نەبوو، ئەمە دەخوێنرێتەوە |

### نیشستی نە-چوونەژوورەوە (Non-Login Shell Session)

| فایل | ناوەرۆک |
|------|---------|
| `/etc/bash.bashrc` | ئیسکریپتی ڕێکخستنی گشتی |
| `~/.bashrc` | فایلی دەستپێکردنی تاکەکەس |

بە بنچینە `~/.bash_profile` بە شێوەیەک نووسراوە کە `~/.bashrc` یش بانگ دەکات:

```bash
# لە ~/.bash_profile
if [ -f ~/.bashrc ]; then
    . ~/.bashrc
fi
```

---

## ناوەرۆکی فایلی دەستپێکردن

نموونەیەکی `~/.bashrc` ی ئاسایی:

```bash
# Source global definitions
if [ -f /etc/bashrc ]; then
    . /etc/bashrc
fi

# ناوی جێگریەوە و ئەمرازی تاکەکەس
alias ll='ls -l --color=auto'
alias la='ls -la --color=auto'
alias l.='ls -d .* --color=auto'

# ژینگەی تاکەکەس
PATH=$PATH:$HOME/bin
export PATH
```

هێڵەکانی بە `#` دەستپێکراو **تێبینی** (comment) ن و جێبەجێ نادەکرێن.

---

## گۆڕینی ژینگە
### Modifying the Environment

**یاسای گشتی:** بۆ زیادکردنی پووشپەر بۆ `PATH` یان پێناسەکردنی گۆڕاوی زیاتر، گۆڕانکاری لە `~/.bash_profile` بکە. بۆ تری، `~/.bashrc` گونجاوترە.

### دەستکاریکردنی فایلی دەستپێکردن

**ئاگادار بە!** پێش دەستکاریکردن، نسخەی پشتیوانی بکە:

```bash
cp ~/.bashrc ~/.bashrc.bak
```

ئێستا بە دەستکاری‌کەر کردەوە:

```bash
nano ~/.bashrc
```

### نموونە: زیادکردنی پووشپەر بۆ PATH

```bash
export PATH=$PATH:$HOME/bin
```

ئەم هێڵە پووشپەری `~/bin` زیاددەکات بۆ `PATH` — پووشپەرێک کە ئیسکریپتەکانی تاکەکەسمان تێدا دادەمەزرێن.

### چالاککردنی گۆڕانکاریەکان

دوای دەستکاریکردن، بۆ جێبەجێکردنی گۆڕانکاریەکان بەبێ دەرچوون و چوونەژوورەوەی دووبارە:

```bash
source ~/.bashrc
```

یان بە کورتی:

```bash
. ~/.bashrc
```

---

## کورتە

تێگەیشتنی ژینگەی شێل بۆ سفارشیکردنی ئەزموونی خێزانی فەرمان پێویستییەکی بنچینەییە. بە `~/.bashrc` و `~/.bash_profile` دەتوانین ناوی جێگریەوە، گۆڕاو و ئەمرازی هەمیشەیی بچەسپێنین کە لە هەر نیشستدا بەردەستن.
