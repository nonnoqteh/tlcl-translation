# بابەتی ٢ — گەڕان
### Chapter 2 – Navigation

---

یەکەم شت کە پێویستە فێربین: چۆن لە سیستەمی فایل (file system) ی لینوکس بگەڕێیت. لە ئەم بابەتەدا، فەرمانەکانی خوارەوە فێردەبین:

- `pwd` — نیشاندانی پووشپەری کاری ئێستا
- `cd` — گۆڕینی پووشپەر
- `ls` — فێرست کردنی ناوەرۆکی پووشپەر

---

## درەختی سیستەمی فایل
### Understanding the File System Tree

لینوکس سیستەمی فایلەکەی بە شێوەی **درەخت** (tree) ڕێکخستووە — پووشپەری کۆکی (root directory) بە `/` دەناسرێت، و هەموو شتێک لە ژێرەوەی دادەمەزرێت.

> **تێبینی گرنگ:** بەپێچەوانەی ویندۆز، لینوکس هیچ نامەی درایڤی (C: یان D:) نەناسێت. هەموو پووشپەرەکان لە ژێر یەک کۆکدا دەژین.

---

## `pwd` — نیشاندانی پووشپەری ئێستا
### pwd – Print Working Directory

```bash
pwd
```

```
/home/me
```

---

## `ls` — فێرست کردنی ناوەرۆک
### ls – List Directory Contents

```bash
ls
```

```
Desktop  Documents  Music  Pictures  Public  Templates  Videos
```

---

## `cd` — گۆڕینی پووشپەر
### cd – Change Directory

```bash
cd /usr/bin
pwd
```

```
/usr/bin
```

```bash
ls
```

```
[...فێرستی درێژی بەرنامەکان...]
```

### ڕێچکەی تەواو (Absolute Pathname)

ڕێچکەیەک کە لە ریشەوە دەستیپێدەکات:

```bash
cd /usr/bin
cd /etc
cd /home/me/Desktop
```

### ڕێچکەی نیسبی (Relative Pathname)

ڕێچکەیەک کە لە پووشپەری ئێستاوە دەستیپێدەکات:

```bash
cd Documents
cd ./Documents
cd ..             # یەک پووشپەر سەرەوە
```

### کورتکوتی گرنگ

| فەرمان | کارکردن |
|--------|--------|
| `cd` | کۆچ بۆ پووشپەری ماڵ |
| `cd -` | کۆچ بۆ پووشپەری پێشوو |
| `cd ~user` | کۆچ بۆ پووشپەری ماڵی بەکارهێنەری تر |
| `cd ..` | کۆچ یەک پووشپەر سەرەوە |

---

## ئاگاداری ناوی فایل

- لینوکس **هەستیار بە گەورەی-بچووکی پیتە** (case-sensitive): `File.txt` و `file.txt` جیاوازن
- ناوی فایل بە `.` دەستپێکردن = **فایلی شاراوە** (hidden file): `.bashrc`
- پووشپەری بەکارهێنان: **فاسیلە بەکارمەهێنە**؛ ئەگەر پێویست بوو، دووقۆل بەکارهێنە: `"My Documents"`

---

## کورتە

لەم بابەتەدا ئامرازە بنچینەیییەکان فێربوین: `pwd` بۆ زانینی شوێنمان، `ls` بۆ بینینی ناوەرۆکی پووشپەر، و `cd` بۆ گەڕانەوە. ئەمانە بنچینەی هەموو کاری خێزانی فەرمان دەگرنەوە.
