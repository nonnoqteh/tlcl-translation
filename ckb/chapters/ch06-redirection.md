# بابەتی ٦ — ئاڕاستەکردنەوە (Redirection)
### Chapter 6 – Redirection

---

لەم بابەتەدا یەکێک لە باشترین تایبەتمەندییەکانی خێزانی فەرمان فێردەبین: **ئاڕاستەکردنەوەی ئینپوت/ئاوتپوت** (I/O Redirection). فەرمانەکانی نوێ:

- `cat` — پەیوەندیکردنی فایلەکان
- `sort` — ڕیزکردنی هێڵەکانی دەق
- `uniq` — لابردنی هێڵی دووبارە
- `grep` — چاپکردنی هێڵی لێکدانەوەی ئالگۆ
- `wc` — ژماردنی هێڵ، وشە و بایت
- `head` — نیشاندانی سەرەتای فایل
- `tail` — نیشاندانی کۆتایی فایل
- `tee` — خوێندنەوە لە ئینپوت و نووسین بۆ ئاوتپوت و فایل

---

## ئینپوت، ئاوتپوت و هەڵەی ستاندارد

زۆری بەرنامەکان دوو جۆر ئاوتپوت بەرهەمدەهێنن:
- **ئەنجامی بەرنامە** — داتای بەرهەمهاتووی بەرنامە
- **پەیامی دۆخ و هەڵە**

لینوکس ئاوتپوتی بەرنامەکان بۆ **ئاوتپوتی ستاندارد** (stdout) دەنێرێت و پەیامی هەڵە بۆ **هەڵەی ستاندارد** (stderr). بە بنچینە هەردوو لە شاشەدا دەردەکەون.

ئینپوت لە **ئینپوتی ستاندارد** (stdin) دێت کە بە بنچینە بە کیبۆرد بەستراوەتەوە.

---

## ئاڕاستەکردنەوەی ئاوتپوتی ستاندارد

بۆ ئاڕاستەکردنەوەی ئاوتپوت بۆ فایل، `>` بەکاردێت:

```bash
ls -l /usr/bin > ls-output.txt
ls -l ls-output.txt
```

```
-rw-rw-r-- 1 me me 167878 2025-02-01 15:07 ls-output.txt
```

`>` هەمیشە فایل لە سەرەوە دەنووسێتەوە. بۆ **زیادکردن** بۆ کۆتایی فایل، `>>` بەکاردێت:

```bash
ls -l /usr/bin >> ls-output.txt
```

---

## ئاڕاستەکردنەوەی هەڵەی ستاندارد

بۆ ئاڕاستەکردنەوەی هەڵە، ژمارەی `2` بەکاردێت:

```bash
ls -l /bin/usr 2> ls-error.txt
```

بۆ ئاڕاستەکردنەوەی هەردووک بۆ یەک فایل:

```bash
# شێوازی کۆن
ls -l /bin/usr > ls-output.txt 2>&1

# شێوازی نوێ (bash نوێ)
ls -l /bin/usr &> ls-output.txt
```

---

## دووری خستنی ئاوتپوتی نەخوازراو

```bash
ls -l /bin/usr 2> /dev/null
```

> `/dev/null` فایلێکی تایبەتییە کە هەر شتێکی پێ بنێرێت لەدەستدەدرێت — «قوزی ڕوونکردنەوەی» یونیکس!

---

## `cat` — پەیوەندیکردنی فایل

```bash
cat ls-output.txt
```

بۆ تێکەڵکردنی فایلە تۆکراوەکان:

```bash
cat movie.mpeg.0* > movie.mpeg
```

ئەگەر `cat` بەبێ ئارگومان بمێنێتەوە، لە ئینپوتی ستاندارد دەخوێنێت:

```bash
cat > lazy_dog.txt
The quick brown fox jumps over the lazy dog.
```

(Ctrl-d بۆ کۆتایی)

بۆ ئاڕاستەکردنەوەی ئینپوتی ستاندارد:

```bash
cat < lazy_dog.txt
```

---

## لووڵەکەشی (Pipelines)

بە عەملگەری `|` ئاوتپوتی فەرمانێک بۆ ئینپوتی فەرمانی داهاتوو دەبەستین:

```bash
ls -l /usr/bin | less
```

> **جیاوازی `>` و `|`:**
> - `>` فەرمانێک بۆ **فایل** دەبەستێت
> - `|` ئاوتپوتی فەرمانێک بۆ **ئینپوتی فەرمانی داهاتوو** دەبەستێت
>
> مەترسی: `command1 > command2` — ئەمە فایلێک بە ناوی `command2` دروستدەکات!

---

## فیلتەرەکان
### Filters

### `sort` — ڕیزکردن

```bash
ls /bin /usr/bin | sort | less
```

### `uniq` — لابردنی دووبارە

```bash
ls /bin /usr/bin | sort | uniq | less
```

### `wc` — ژماردن

```bash
wc ls-output.txt
```

```
 7902  64566 503634 ls-output.txt
```

```bash
ls /bin /usr/bin | sort | uniq | wc -l
```

### `grep` — گەڕانی ئالگۆ

```bash
ls /bin /usr/bin | sort | uniq | grep zip
```

```
bunzip2
bzip2
gunzip
gzip
...
```

**ئۆپشنەکانی بەسوود:**
- `-i` — بەبێ ئاگاداری گەورە/بچووک
- `-v` — تەنها هێڵی **نالێکدانەوە**
- `-l` — تەنها ناوی فایل

### `head` و `tail`

```bash
head -n 5 ls-output.txt    # ٥ هێڵی یەکەم
tail -n 5 ls-output.txt    # ٥ هێڵی کۆتایی
tail -f /var/log/messages   # شوێنکردنی زیندوو
```

### `tee`

```bash
ls /usr/bin | tee ls.txt | grep zip
```

## لینوکس بریتییە لە خەیاڵ

کاتێک لێ دەپرسن جیاوازی ویندۆز و لینوکس چییە، وێنەی یاریچاکێکم بەکار دەهێنم.

ویندۆز مانەندی گیم-بۆیێکە — دەیکڕێیت، دەیکەیتەوە و یاریی دەکەیت. بەڵام دواتر ستووخۆش دەبیت. دەڕۆیت یاری نوێ بکڕیت بەڵام دەڵێن ئەوەی دەتەوێت نییە چونکە «داواکاری بازاڕ نییە».

لینوکس، بەپێچەوانەوە، مانەندی گەورەترین کۆمەڵی قاچیانە دروستکردنی دنیایەتی. دەیکەیتەوە و کۆمەڵی ئەزیمی قاچ دەبینیت. دەستی دروستکردن دەکەیت — یەکەم ئەوانەی پێشنیار کراوە، پاشان ئایدیای خۆت. هیچکات پێویست ناکات بگەڕێیتەوە بۆ دوکان. کۆمەڵ شێوەی خەیاڵی خۆت دەگرێت.

## کورتە

هەمیشە مستەنداتی هەر فەرمانێک لەم بابەتەدا فێربووی بخوێنەوە. زیاترین بەکارهێنانی بنچینەیی بینیوە، بەڵام ئۆپشنی زۆر جالبیان هەیە. هەرچەند زیاتر لەگەڵ لینوکس کار بکەیت، دەبینیت تواناکانی ئاڕاستەکردنەوە بۆ چارەسەرکردنی کێشەی تایبەت سووپەرن.
