# وەرگێڕانی کوردی-سۆرانی — هێڵی فەرمانی لینوکس

ئەم بوخچەیە وەرگێڕانی کوردی-سۆرانی باپەڕەکانی کتێبەکەی تێدایە.

## ڕێنمایی وەرگێڕ

- هەر باپەڕەیەک فایلێکی جیاوازی لە `chapters/` ہەیە
- ناوی فایلەکان: `ch01-shell-chiiye.md`، `ch02-geran.md` و ...
- لە سەرەتای هەر فایلێکدا، سەردێڕی باپەڕەکە بە کوردی و ئینگلیزی بنووسرێت
- کۆدەکانی شێل و فەرمانەکان **وەرنەگێڕدرێن**، تەنها ڕوونکردنەوەکانیان
- پەیوەندیدارانی تەکنیکی: یەکەم هاوتای کوردی، پاشان ئینگلیزی لە کەوانەدا — بۆ نموونە: `بوخچە (directory)`

## فەرهەنگی پرکار

| ئینگلیزی | کوردی-سۆرانی |
|----------|--------------|
| shell | شێل |
| command | فەرمان |
| directory | بوخچە |
| file | فایل |
| process | پرۆسە |
| permission | مۆڵەت |
| terminal | تەرمینال |
| prompt | داخوازی |
| argument | ئارگیومێنت |
| option / flag | هەڵبژاردە |
| standard input | دەستپێکردنی ستانداردی |
| standard output | دەرچوونی ستانداردی |
| pipeline | بۆری |
| redirection | ئاڕاستەکردنەوە |
| environment variable | گۆڕاوی ژینگە |
